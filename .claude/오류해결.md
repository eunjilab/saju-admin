# 오류 해결 패턴 모음

## 서버 관련

### 포트 충돌 (Address already in use)
**증상**: 서버 실행 시 "Address already in use" 또는 연결 실패
**해결**:
```bash
pkill -f "python3 -m http.server 8080"  # 기존 서버 종료
# 또는
lsof -i :8080  # 포트 사용 중인 프로세스 확인
kill -9 [PID]  # 해당 프로세스 종료
```

## Git 관련

### push 거부됨 (rejected)
**증상**: `git push` 시 rejected 오류
**해결**:
```bash
git pull origin main --rebase  # 원격 변경사항 먼저 가져오기
git push origin main
```

### 충돌 발생 (conflict)
**증상**: merge conflict 메시지
**해결**:
1. 충돌 파일 열어서 `<<<<<<<` 부분 수동 수정
2. `git add .` → `git commit`

## Netlify 관련

### 배포 실패
**증상**: Netlify deploy 실패
**해결**:
1. `netlify status` 로 로그인 상태 확인
2. `netlify login` 으로 재로그인
3. 다시 `npx netlify deploy --prod`

# 신살 해석 가이드

신살(神殺)은 사주에서 특별한 기운을 나타내는 요소입니다.
길신(吉神), 흉신(凶神), 양면신으로 구분됩니다.

---

## 1. 신살 분류

### 길신 (吉神) - 좋은 기운
천을귀인, 월덕귀인, 천덕귀인, 문창귀인, 문곡귀인, 학당귀인,
금여록, 천주귀인, 복성귀인, 태극귀인, 삼기귀인, 건록, 암록,
일덕, 일귀, 천사일, 십령일

### 흉신 (凶神) - 주의할 기운
겁살, 재살, 천살, 지살, 월살, 망신살, 양인살, 괴강살, 백호살,
탕화살, 천라지망, 십악대패일, 급각살, 비인살, 고란살, 사패

### 양면신 (兩面神) - 상황에 따라 길흉이 달라짐
도화살, 역마살, 화개살, 홍염살, 현침살, 공망, 귀문관살,
고진살, 과숙살, 원진살, 낙정관살

## 4. 주요 양면신 상세

### 도화살 (桃花殺)
**매력과 이성운. 길하면 인기, 흉하면 스캔들**

| 기준 (년지/일지) | 도화 |
|:---:|:---:|
| 申子辰 | 酉 |
| 寅午戌 | 卯 |
| 巳酉丑 | 午 |
| 亥卯未 | 子 |

**길한 경우:**
- 연예인, 예술가에게 유리
- 대인관계 좋음
- 매력적, 사교적

**흉한 경우:**
- 이성 문제, 바람
- 구설수, 스캔들
- 음주/향락 문제

### 역마살 (驛馬殺)
**이동과 변화. 길하면 해외운, 흉하면 방랑**

| 기준 (년지/일지) | 역마 |
|:---:|:---:|
| 申子辰 | 寅 |
| 寅午戌 | 申 |
| 巳酉丑 | 亥 |
| 亥卯未 | 巳 |

**길한 경우:**
- 해외 진출 유리
- 무역, 여행업 적합
- 활동적 직업 좋음

**흉한 경우:**
- 정착 못함
- 직장/주거 불안정
- 떠돌이 생활

### 화개살 (華蓋殺)
**예술과 영성. 고독하지만 재능 있음**

| 기준 (년지/일지) | 화개 |
|:---:|:---:|
| 申子辰 | 辰 |
| 寅午戌 | 戌 |
| 巳酉丑 | 丑 |
| 亥卯未 | 未 |

**특징:**
- 예술적 재능
- 종교/철학 관심
- 고독을 즐김
- 영적 감각 발달

### 공망 (空亡)
**비어있는 기운. 노력해도 결과가 늦음**

| 일주 旬 | 공망 |
|:---:|:---:|
| 甲子旬 | 戌, 亥 |
| 甲戌旬 | 申, 酉 |
| 甲申旬 | 午, 未 |
| 甲午旬 | 辰, 巳 |
| 甲辰旬 | 寅, 卯 |
| 甲寅旬 | 子, 丑 |

**해석:**
- 해당 지지의 기운이 약해짐
- 관련 육친과의 인연 약함
- 노력의 결과가 늦게 옴
- 충(沖)이나 합(合)으로 공망이 풀릴 수 있음

## 7. 신살 조합 해석

### 길신이 많을 때
- 위기에서 도움을 받음
- 인덕이 많음
- 흉살을 제압

### 흉신이 많을 때
- 건강/사고 주의
- 인간관계 문제 발생 가능
- 길신이 있으면 완화됨

### 양면신 해석 기준
- **길신과 함께** → 좋은 방향으로
- **흉신과 함께** → 나쁜 방향으로
- **단독** → 상황에 따라 달라짐

## 8. 해석 주의사항

1. **신살만으로 판단 금지**: 사주 전체와 함께 봐야 함
2. **용신과 연결**: 용신 위치에 길신 = 대길, 기신 위치에 흉신 = 대흉
3. **흉신 과대 해석 금지**: 흉신도 상황에 따라 좋을 수 있음
4. **현대적 해석**: 과거와 현대의 직업/생활이 다름을 고려
5. **귀인 중시**: 천을귀인, 천덕귀인, 월덕귀인은 흉살 제압 효과

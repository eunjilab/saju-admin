# 십성 해석 가이드

십성(十星)은 일간을 기준으로 다른 천간/지지와의 관계를 나타냅니다.

---

## 1. 십성 계산 원리

### 핵심 규칙
**일간(日干)을 기준으로 다른 글자와의 오행 관계 + 음양 일치 여부로 결정**

| 관계 | 같은 음양 | 다른 음양 |
|:---:|:---:|:---:|
| 나와 같은 오행 | 비견(比肩) | 겁재(劫財) |
| 내가 생하는 오행 | 식신(食神) | 상관(傷官) |
| 내가 극하는 오행 | 편재(偏財) | 정재(正財) |
| 나를 극하는 오행 | 편관(偏官) | 정관(正官) |
| 나를 생하는 오행 | 편인(偏印) | 정인(正印) |

## 3. 십성 5그룹 상세 해석

### 3-1. 비겁 (比劫) - 나와 같은 오행

**비견 (比肩)** - 나와 같은 오행, 같은 음양
- 의미: 나와 동등한 존재, 형제, 친구, 동료, 경쟁자
- 긍정: 자립심, 독립심, 추진력, 자존심
- 부정: 고집, 아집, 분리, 경쟁, 재물 손실
- 관계: 남녀 모두 - 형제자매, 동료, 라이벌

**겁재 (劫財)** - 나와 같은 오행, 다른 음양
- 의미: 나의 재물을 빼앗는 존재
- 긍정: 활동력, 사교성, 승부욕, 배포
- 부정: 재물 손실, 배신, 투기, 도박성
- 관계: 이성 형제, 경쟁자, 사업 파트너

### 3-2. 식상 (食傷) - 내가 생하는 오행

**식신 (食神)** - 내가 생하는 오행, 같은 음양
- 의미: 먹을 복, 표현력, 재능, 자녀(여성)
- 긍정: 낙천적, 식복, 예술성, 표현력, 건강
- 부정: 게으름, 나태, 향락, 비만
- 관계: 여성-첫째 딸, 남성-장모/외조모

**상관 (傷官)** - 내가 생하는 오행, 다른 음양
- 의미: 관(官)을 상하게 함, 반항, 비판
- 긍정: 창의력, 예술성, 언변, 기술, 전문성
- 부정: 반항심, 비판, 관재, 이혼수 (여성)
- 관계: 여성-첫째 아들, 남성-외할아버지

### 3-3. 재성 (財星) - 내가 극하는 오행

**편재 (偏財)** - 내가 극하는 오행, 같은 음양
- 의미: 유동적 재물, 사업, 투자, 아버지
- 긍정: 사업수완, 융통성, 사교성, 재테크
- 부정: 투기, 낭비, 바람기, 건강 문제
- 관계: 남녀 모두-아버지, 남성-정부/애인

**정재 (正財)** - 내가 극하는 오행, 다른 음양
- 의미: 고정적 재물, 월급, 정직한 재산
- 긍정: 성실함, 검소함, 저축, 안정적 수입
- 부정: 인색함, 융통성 부족, 소심함
- 관계: 남성-아내, 여성-시아버지

### 3-4. 관성 (官星) - 나를 극하는 오행

**편관 (偏官/七殺)** - 나를 극하는 오행, 같은 음양
- 의미: 권위, 권력, 무관, 스트레스
- 긍정: 리더십, 결단력, 추진력, 명예
- 부정: 스트레스, 질병, 소송, 관재
- 관계: 남성-첫째 아들, 여성-정부/애인

**정관 (正官)** - 나를 극하는 오행, 다른 음양
- 의미: 정당한 권위, 직장, 명예, 법
- 긍정: 품위, 명예, 안정적 직장, 책임감
- 부정: 압박감, 규제, 융통성 부족
- 관계: 남성-첫째 딸, 여성-남편

### 3-5. 인성 (印星) - 나를 생하는 오행

**편인 (偏印/梟神)** - 나를 생하는 오행, 같은 음양
- 의미: 비정규 학문, 종교, 철학, 특수 기술
- 긍정: 직관력, 창의력, 예술성, 영성
- 부정: 게으름, 의존성, 불안정, 외로움
- 관계: 남녀 모두-계모, 유모, 스승

**정인 (正印)** - 나를 생하는 오행, 다른 음양
- 의미: 정규 학문, 어머니, 문서, 자격증
- 긍정: 학업, 명예, 도움, 보호, 인덕
- 부정: 의존성, 나태, 게으름, 우유부단
- 관계: 남녀 모두-어머니, 스승

## 4. 십성 조합 해석

### 좋은 조합

| 조합 | 의미 |
|:---|:---|
| 식신 + 정재 | 식신생재 - 재능으로 돈을 벎 |
| 정관 + 정인 | 관인상생 - 명예와 학업 모두 성공 |
| 상관 + 편재 | 상관생재 - 기술/재능으로 큰 돈 |
| 편관 + 정인 | 살인상생 - 권력을 지혜로 다스림 |

### 주의할 조합

| 조합 | 의미 |
|:---|:---|
| 상관 + 정관 | 상관견관 - 관재, 직장 문제 |
| 비견 + 정재 | 비겁쟁재 - 재물 다툼, 손실 |
| 편인 + 식신 | 효신탈식 - 식신의 복을 빼앗김 |
| 겁재 + 정재 | 겁재패재 - 큰 재물 손실 |

## 5. 십성 분포 해석

### 특정 십성이 많을 때

| 십성 | 3개 이상일 때 |
|:---|:---|
| 비겁 과다 | 고집, 독립심 강함, 협동 어려움, 재물 분산 |
| 식상 과다 | 표현력 좋으나 말 많음, 직업 불안정, 건강 주의 |
| 재성 과다 | 현실적, 물질적이나 인색, 어머니와 갈등 |
| 관성 과다 | 스트레스 많음, 직장/규제에 시달림, 건강 주의 |
| 인성 과다 | 생각 많음, 게으름, 의존적, 실행력 부족 |

### 특정 십성이 없을 때

| 십성 | 없을 때 |
|:---|:---|
| 비겁 없음 | 외로움, 형제 인연 약함, 자립해야 함 |
| 식상 없음 | 표현력 부족, 자녀 인연 약함 (여성) |
| 재성 없음 | 재물운 약함, 아버지 인연 약함 |
| 관성 없음 | 직장운 약함, 남편 인연 약함 (여성) |
| 인성 없음 | 학업 인연 약함, 어머니 인연 약함 |

## 6. 해석 예시

### 예: 己土 일간의 십성 분석

```
┌────────┬────────┬────────┬────────┐
│  시주  │  일주  │  월주  │  연주  │
├────────┼────────┼────────┼────────┤
│ 庚(상관)│ 己(일간)│ 丁(편인)│ 甲(정관)│
│ 午(편인)│ 酉(식신)│ 卯(편관)│ 戌(겁재)│
└────────┴────────┴────────┴────────┘

십성 분포:
- 비겁: 1 (戌 중 戊)
- 식상: 2 (庚, 酉)
- 재성: 0
- 관성: 2 (甲, 卯)
- 인성: 2 (丁, 午)

해석:
- 식상이 가장 강함 → 표현력, 재능, 기술력 뛰어남
- 재성 없음 → 재물은 노력으로 만들어야 함
- 관성 2개 → 직장운/명예운 있으나 스트레스도 있음
- 인성 2개 → 학업운 좋음, 어머니 인연 좋음
```

## 7. 주의사항

1. **지지는 지장간으로**: 지지의 십성은 지장간 중 정기(正氣)로 판단
2. **세력 판단**: 단순 개수보다 위치와 관계도 중요
3. **일간 강약 고려**: 신강이면 식상/재성이 길, 신약이면 인성/비겁이 길
4. **상호작용 확인**: 합/충/형/파/해 관계 함께 분석

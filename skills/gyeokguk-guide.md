# 격국/용신 해석 가이드

격국(格局)은 사주의 구조와 특성을 나타내고,
용신(用神)은 사주에서 가장 필요한 오행입니다.

---

## 1. 격국 (格局) 기본 개념

### 격국이란?
- 사주의 전체적인 구조와 성향
- 월지의 지장간 정기(正氣)를 기준으로 결정
- 일간과의 관계로 8격을 판단

### 8격 (정격)

월지 정기의 십성에 따라 8가지 격으로 분류합니다.

| 격 | 월지 정기의 십성 | 기본 성향 |
|:---:|:---:|:---|
| 비견격 | 비견 | 독립심, 자존심, 형제 인연 |
| 겁재격 | 겁재 | 활동적, 승부욕, 분산 경향 |
| 식신격 | 식신 | 식복, 낙천적, 표현력 |
| 상관격 | 상관 | 창의력, 반항심, 기술 |
| 편재격 | 편재 | 사업수완, 융통성, 아버지 |
| 정재격 | 정재 | 성실함, 검소함, 안정 |
| 편관격 | 편관 | 권위, 결단력, 리더십 |
| 정관격 | 정관 | 품위, 명예, 책임감 |
| 편인격 | 편인 | 직관력, 특수재능, 고독 |
| 정인격 | 정인 | 학업, 인덕, 보호 |

## 2. 격국 결정 방법

### 2-1. 월지 정기 확인

| 월지 | 지장간 | 정기 |
|:---:|:---|:---:|
| 子 | 癸 | 癸 |
| 丑 | 癸辛己 | 己 |
| 寅 | 戊丙甲 | 甲 |
| 卯 | 乙 | 乙 |
| 辰 | 乙癸戊 | 戊 |
| 巳 | 戊庚丙 | 丙 |
| 午 | 己丁 | 丁 |
| 未 | 丁乙己 | 己 |
| 申 | 戊壬庚 | 庚 |
| 酉 | 辛 | 辛 |
| 戌 | 辛丁戊 | 戊 |
| 亥 | 戊甲壬 | 壬 |

### 2-2. 일간 기준 십성 판단

월지 정기와 일간의 관계로 격국 결정

**예시: 己土 일간, 卯月生**
```
월지 卯의 정기 = 乙
乙은 己 기준으로 = 편관
→ 편관격(偏官格)
```

## 3. 각 격국 상세 해석

### 식신격 (食神格)
**먹을 복이 있고 낙천적인 격**

- 성격: 낙천적, 여유로움, 베푸는 성격
- 직업: 요식업, 서비스업, 교육, 예술
- 장점: 식복, 대인관계 좋음, 건강
- 단점: 게으름, 나태, 의지 약함
- 길한 운: 재성운 (식신생재)
- 주의할 운: 편인운 (효신탈식)

### 상관격 (傷官格)
**창의력과 반항심이 강한 격**

- 성격: 창의적, 비판적, 기술력
- 직업: 예술가, 기술자, 전문직, 프리랜서
- 장점: 뛰어난 재능, 독창성
- 단점: 반항심, 권위 거부, 관재
- 길한 운: 재성운 (상관생재)
- 주의할 운: 정관운 (상관견관)

### 편재격 (偏財格)
**사업 수완이 있는 격**

- 성격: 융통성, 사교적, 현실적
- 직업: 사업가, 투자, 유통, 무역
- 장점: 재테크, 사업 감각
- 단점: 투기성, 낭비, 바람기
- 길한 운: 식상운 (식상생재)
- 주의할 운: 비겁운 (비겁쟁재)

### 정재격 (正財格)
**성실하고 안정적인 격**

- 성격: 검소함, 성실함, 착실함
- 직업: 회계, 금융, 공무원, 안정적 직장
- 장점: 저축, 신뢰, 안정
- 단점: 인색함, 융통성 부족
- 길한 운: 식상운, 관성운
- 주의할 운: 비겁운 (재물 손실)

### 편관격 (偏官格/七殺格)
**권위와 리더십이 강한 격**

- 성격: 결단력, 카리스마, 강인함
- 직업: 군인, 경찰, 정치, 경영
- 장점: 리더십, 추진력, 권위
- 단점: 스트레스, 건강 문제, 권력욕
- 길한 운: 인성운 (살인상생)
- 주의할 운: 재성운 (재생살)

### 정관격 (正官格)
**품위와 명예가 있는 격**

- 성격: 품위, 책임감, 원칙주의
- 직업: 공무원, 대기업, 법조계, 관료
- 장점: 명예, 안정, 신뢰
- 단점: 융통성 부족, 규제에 얽매임
- 길한 운: 재성운 (재생관), 인성운
- 주의할 운: 상관운 (상관견관)

### 편인격 (偏印格/梟神格)
**특수한 재능이 있는 격**

- 성격: 직관력, 예민함, 고독
- 직업: 종교, 철학, 예술, 특수기술
- 장점: 영적 감각, 독창성
- 단점: 의존성, 불안정, 고독
- 길한 운: 비겁운, 관성운
- 주의할 운: 재성운 (인성 약화)

### 정인격 (正印格)
**학업과 인덕이 있는 격**

- 성격: 학구적, 온화함, 보수적
- 직업: 교육, 학문, 공무원, 문서직
- 장점: 학업 성취, 인덕, 보호
- 단점: 의존성, 게으름, 우유부단
- 길한 운: 관성운 (관인상생)
- 주의할 운: 재성운 (재파인)

## 4. 용신 (用神)

### 용신이란?
- 사주에서 가장 필요한 오행
- 일간의 균형을 맞추는 기준
- 용신 운이 좋은 운, 기신 운이 나쁜 운

### 4신 (四神) 체계

| 명칭 | 역할 | 운 영향 |
|:---:|:---|:---|
| 용신 (用神) | 가장 필요한 오행 | 최고의 운 |
| 희신 (喜神) | 용신을 돕는 오행 | 좋은 운 |
| 기신 (忌神) | 용신을 해치는 오행 | 나쁜 운 |
| 구신 (仇神) | 기신을 돕는 오행 | 매우 나쁜 운 |

---

## 5. 신강/신약 판단

### 일간 강약 판단 요소

1. **월령 (月令)** - 가장 중요
   - 월지가 일간을 생하면 신강
   - 월지가 일간을 극하면 신약

2. **통근 (通根)**
   - 지지에 일간의 뿌리가 있으면 신강
   - 예: 甲일간에 寅, 卯가 있으면 통근

3. **비겁/인성 개수**
   - 많으면 신강
   - 적으면 신약

4. **생조 (生助)**
   - 생하고 도와주는 글자 개수

### 신강/신약 판단표

| 조건 | 신강 | 신약 |
|:---|:---:|:---:|
| 월령이 일간을 생/비 | ✓ | |
| 비겁 2개 이상 | ✓ | |
| 인성 2개 이상 | ✓ | |
| 지지에 통근 | ✓ | |
| 월령이 일간을 극/설 | | ✓ |
| 관성/재성 과다 | | ✓ |
| 식상 과다 | | ✓ |

## 6. 용신 선정 방법

### 신강 사주의 용신

신강하면 일간의 기운을 빼주는 오행이 용신

| 우선순위 | 용신 | 이유 |
|:---:|:---:|:---|
| 1순위 | 관성 | 일간을 제어 |
| 2순위 | 재성 | 인성을 제어, 관성 생 |
| 3순위 | 식상 | 일간을 설기 |

**신강 사주 4신 예시:**
```
일간: 己土 (신강)
- 용신: 水 (재성) - 일간을 극
- 희신: 木 (관성) - 용신을 생
- 기신: 火 (인성) - 일간을 더 강하게
- 구신: 土 (비겁) - 기신을 생
```

### 신약 사주의 용신

신약하면 일간을 도와주는 오행이 용신

| 우선순위 | 용신 | 이유 |
|:---:|:---:|:---|
| 1순위 | 인성 | 일간을 생 |
| 2순위 | 비겁 | 일간을 도움 |

**신약 사주 4신 예시:**
```
일간: 己土 (신약)
- 용신: 火 (인성) - 일간을 생
- 희신: 土 (비겁) - 일간과 같음
- 기신: 木 (관성) - 일간을 극
- 구신: 水 (재성) - 용신을 극
```

## 7. 특수 격국

### 종격 (從格)
일간이 너무 약해서 강한 쪽을 따라가는 격

| 종격 | 조건 | 용신 |
|:---:|:---|:---:|
| 종살격 | 관성이 압도적 | 관성 |
| 종재격 | 재성이 압도적 | 재성 |
| 종아격 | 식상이 압도적 | 식상 |
| 종강격 | 비겁/인성이 압도적 | 비겁/인성 |

### 특수 격국
| 격 | 조건 | 특징 |
|:---:|:---|:---|
| 건록격 | 월지가 건록 | 자립심 강함 |
| 양인격 | 월지가 양인 | 강인함, 승부욕 |
| 잡기격 | 월지가 辰戌丑未 | 다재다능 |

## 8. 격국/용신 해석 예시

### 예: 己土 일간, 卯月生, 신강

```
사주:
┌────────┬────────┬────────┬────────┐
│   庚   │   己   │   丁   │   甲   │
│   午   │   酉   │   卯   │   戌   │
└────────┴────────┴────────┴────────┘

1. 격국 판단
- 월지 卯의 정기 = 乙
- 乙은 己 기준 = 편관
- 격국: 편관격

2. 신강/신약 판단
- 월령: 卯(목) → 己를 극함 (약화 요소)
- 비겁: 戌 중 戊 (1개)
- 인성: 丁, 午 (2개) → 강화 요소
- 결론: 인성 도움으로 신강

3. 용신 선정 (신강)
- 용신: 水 (재성) - 부족한 오행, 일간 제어
- 희신: 木 (관성) - 용신 생, 일간 제어
- 기신: 火 (인성) - 이미 많음, 더 강해지면 안됨
- 구신: 土 (비겁) - 기신 생

4. 해석
- 편관격 + 신강 = 리더십 있고 결단력 강함
- 용신 수(水) = 지혜, 유연성, 재물 필요
- 인성 과다 = 생각 많음, 행동력 필요
- 재성 부재 = 재물은 노력으로
```

## 9. 주의사항

1. **격국만으로 판단 금지**: 전체 사주 구조 함께 봐야 함
2. **신강/신약 신중히**: 경계선에 있는 경우 주의
3. **용신은 상황에 따라**: 시기, 환경에 따라 변할 수 있음
4. **대운과 연결**: 용신운이 언제 오는지 확인
5. **종격 주의**: 종격인지 일반격인지 구분 중요

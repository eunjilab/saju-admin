# 병존/반복 구조 해석 가이드

같은 글자가 여러 개 있거나 오행이 편중된 경우의 해석 가이드입니다.

---

## 1. 천간 병존 (天干竝存)

같은 천간이 2개 이상 있는 경우입니다.

### 양간(陽干) 병존

| 병존 | 의미 |
|:---:|:---|
| 甲甲 | 리더십 강함, 고집 셈, 타협 어려움, 나무가 너무 빽빽 |
| 丙丙 | 열정 과다, 급한 성격, 화재/열병 주의, 눈에 띔 |
| 戊戊 | 신뢰감 높음, 고집 셈, 융통성 부족, 산이 겹침 |
| 庚庚 | 결단력 강함, 냉정함, 무뚝뚝, 쇠가 너무 강함 |
| 壬壬 | 지혜로움, 생각 많음, 우유부단할 수 있음, 물이 넘침 |

### 음간(陰干) 병존

| 병존 | 의미 |
|:---:|:---|
| 乙乙 | 유연함, 적응력, 의지 약할 수 있음, 덩굴이 엉킴 |
| 丁丁 | 섬세함, 예민함, 신경질적, 촛불이 두 개 |
| 己己 | 포용력, 현실적, 욕심 많을 수 있음 |
| 辛辛 | 예민함, 날카로움, 까칠함, 보석이 두 개 |
| 癸癸 | 직관력, 감수성, 우울할 수 있음, 비가 계속 옴 |

### 천간 병존 해석 원칙

1. **같은 기운 증폭**: 해당 오행의 특성이 2배
2. **비겁 과다**: 일간과 같은 글자면 비겁 2개 → 고집, 독립심 강함
3. **쟁합 가능성**: 합 대상이 있으면 쟁합 상황 발생

## 2. 지지 병존 (地支竝存)

같은 지지가 2개 이상 있는 경우입니다.

### 지지 병존 + 자형(自刑)

자형이 되는 지지가 2개 있으면 특히 주의가 필요합니다.

| 병존 | 자형 여부 | 의미 |
|:---:|:---:|:---|
| 子子 | ❌ | 수(水) 과다, 감정 기복, 신장/방광 주의 |
| 丑丑 | ❌ | 토(土) 과다, 고집, 소화기 주의 |
| 寅寅 | ❌ | 목(木) 과다, 추진력, 간/담 주의 |
| 卯卯 | ❌ | 목(木) 과다, 유연함, 간 주의 |
| **辰辰** | ⭕ 자형 | 자책, 우울, 창고가 열림 |
| 巳巳 | ❌ | 화(火) 과다, 열정, 심장 주의 |
| **午午** | ⭕ 자형 | 번아웃, 열정 과다, 심장 주의 |
| 未未 | ❌ | 토(土) 과다, 감성적, 소화기 주의 |
| 申申 | ❌ | 금(金) 과다, 결단력, 폐 주의 |
| **酉酉** | ⭕ 자형 | 예민, 자해적 성향, 폐/대장 주의 |
| 戌戌 | ❌ | 토(土) 과다, 고집, 창고 열림 |
| **亥亥** | ⭕ 자형 | 생각 과다, 우울증 주의 |

### 자형(自刑) 상세 해석

자형은 **스스로를 해치는 기운**입니다.

| 자형 | 상세 해석 |
|:---:|:---|
| 辰辰 | 창고가 스스로 열림, 재물 변동, 비밀 노출, 자기 비판 |
| 午午 | 태양이 두 개, 열정 과다로 번아웃, 화병, 심장/혈압 |
| 酉酉 | 날카로움이 배가, 예민/신경질, 자해적 성향, 폐/피부 |
| 亥亥 | 물이 넘침, 생각 과다, 우울/불안, 신장/생식기 |

## 3. 오행 편중 (五行偏重)

특정 오행이 3개 이상 집중된 경우입니다.

### 오행별 과다 해석

| 오행 | 과다 시 특징 | 주의 건강 | 부족 오행 영향 |
|:---:|:---|:---|:---|
| 木 과다 | 성급, 분노, 계획만 많음 | 간/담, 근육 | 금(金) 부족 → 결단력 약함 |
| 火 과다 | 열정적, 급함, 감정적 | 심장/소장 | 수(水) 부족 → 지혜 부족 |
| 土 과다 | 고집, 느림, 의심 많음 | 비장/위장 | 목(木) 부족 → 추진력 약함 |
| 金 과다 | 냉정, 비판적, 융통성 없음 | 폐/대장 | 화(火) 부족 → 열정 부족 |
| 水 과다 | 우유부단, 두려움, 감정적 | 신장/방광 | 토(土) 부족 → 안정감 부족 |

### 오행 완전 부재 (0개)

| 부재 오행 | 의미 | 보완 방법 |
|:---:|:---|:---|
| 木 없음 | 시작/계획 약함, 간 주의 | 푸른색, 동쪽, 봄 활용 |
| 火 없음 | 열정/표현 약함, 심장 주의 | 붉은색, 남쪽, 여름 활용 |
| 土 없음 | 안정/신뢰 약함, 위장 주의 | 노란색, 중앙, 환절기 활용 |
| 金 없음 | 결단/실행 약함, 폐 주의 | 흰색, 서쪽, 가을 활용 |
| 水 없음 | 지혜/유연 약함, 신장 주의 | 검은색, 북쪽, 겨울 활용 |

## 4. 특수 병존 구조

### 4-1. 간지동체 (干支同體)

천간과 지지의 오행이 같은 경우입니다.

| 일주 | 의미 |
|:---:|:---|
| 甲寅 | 목(木) 강함, 리더십, 고집 |
| 乙卯 | 목(木) 강함, 유연, 예술성 |
| 丙午 | 화(火) 강함, 열정, 급함 |
| 丁巳 | 화(火) 강함, 지혜, 예민 |
| 戊辰/戊戌 | 토(土) 강함, 신뢰, 고집 |
| 己丑/己未 | 토(土) 강함, 포용, 현실적 |
| 庚申 | 금(金) 강함, 결단, 냉정 |
| 辛酉 | 금(金) 강함, 예민, 날카로움 |
| 壬子 | 수(水) 강함, 지혜, 감정적 |
| 癸亥 | 수(水) 강함, 직관, 우울 |

### 4-2. 양인좌(羊刃坐) 일주

일지에 양인이 있는 강력한 일주입니다.

| 일주 | 양인 | 특징 |
|:---:|:---:|:---|
| 甲卯 | 卯 | 목(木) 극강, 고집 매우 셈 |
| 丙午 | 午 | 화(火) 극강, 열정적, 급함 |
| 戊午 | 午 | 화생토, 열정+안정 |
| 庚酉 | 酉 | 금(金) 극강, 결단력 매우 강함 |
| 壬子 | 子 | 수(水) 극강, 지혜+감정 |

## 5. 병존 + 공망 조합

### 공망된 지지가 2개일 때

공망(空亡)된 글자가 병존하면 특수한 해석이 필요합니다.

| 상황 | 해석 |
|:---|:---|
| 공망 지지가 2개 | 공망 효과 약해짐 (서로 채워줌) |
| 공망 + 충 | 공망이 풀림 (충공) |
| 공망 + 합 | 공망이 풀림 (합공) |

## 6. 쌍충/쌍합 구조

### 쌍충 (같은 충이 2번)

```
예: 子 - 午 - 子 - 午
    ↑───↓   ↑───↓
    子午충   子午충
```

**해석:**
- 충의 효과가 매우 강함
- 변동/이동이 잦음
- 감정 기복 심함
- 해당 오행 관련 건강 주의

### 쌍합 (같은 합이 2번)

```
예: 子 - 丑 - 子 - 丑
    ↑───↓   ↑───↓
    子丑합   子丑합
```

**해석:**
- 합의 효과가 매우 강함
- 인연/결합 기운 강함
- 해당 합화 오행이 매우 강해짐

## 7. 해석 예시

### 예시 1: 甲戌 / 丁卯 / 己酉 / 庚午

```
오행 분포:
- 목(木): 甲, 卯 = 2개
- 화(火): 丁, 午 = 2개
- 토(土): 己, 戌 = 2개
- 금(金): 庚, 酉 = 2개
- 수(水): 0개 ⚠️

분석:
- 수(水) 완전 부재 → 지혜, 유연성 부족 가능
- 4개 오행이 균등하게 분포 (각 2개)
- 수(水)를 보완해야 함
```

### 예시 2: 壬子 / 壬子 / 壬子 / 甲寅

```
병존 구조:
- 壬 3개 (천간 삼병존)
- 子 3개 (지지 삼병존)

오행 분포:
- 수(水): 6개 (壬3 + 子3) = 극과다
- 목(木): 2개 (甲 + 寅)

분석:
- 수(水) 극과다 → 매우 지혜로우나 우유부단
- 감정 기복 심함, 신장/방광 건강 주의
- 목(木)이 있어 수생목으로 설기(泄氣)됨
- 화(火), 토(土) 완전 부재 → 열정, 안정감 부족
```

## 8. 주의사항

1. **단순 개수만 보지 말 것**: 위치와 관계도 중요
2. **계절(월지) 고려**: 월지 계절에 따라 강약 달라짐
3. **용신과 연결**: 과다 오행이 용신인지 기신인지 확인
4. **건강 연관**: 과다/부족 오행의 장기 주의
5. **자형 주의**: 辰辰, 午午, 酉酉, 亥亥는 특별 주의

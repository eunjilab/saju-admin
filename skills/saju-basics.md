## 3. 오행 (五行) - 상생과 상극

### 상생 (서로 돕는 관계) - 시계 방향

```
목(木) → 화(火) → 토(土) → 금(金) → 수(水) → 목(木)
```

- **목생화**: 나무가 타서 불이 됨
- **화생토**: 불이 타고 재가 되어 흙이 됨
- **토생금**: 흙에서 금속이 나옴
- **금생수**: 금속이 차가워지면 물방울이 맺힘
- **수생목**: 물이 나무를 키움

### 상극 (서로 억제하는 관계) - 별 모양

```
목(木) → 토(土) → 수(水) → 화(火) → 금(金) → 목(木)
```

- **목극토**: 나무가 흙의 영양을 빼앗음
- **토극수**: 흙이 물을 막음 (제방)
- **수극화**: 물이 불을 끔
- **화극금**: 불이 쇠를 녹임
- **금극목**: 쇠(도끼)가 나무를 벰

## 5. 사주 구조 (四柱)

사주는 4개의 기둥으로 이루어집니다.

```
┌────────┬────────┬────────┬────────┐
│  시주  │  일주  │  월주  │  연주  │
│ (時柱) │ (日柱) │ (月柱) │ (年柱) │
├────────┼────────┼────────┼────────┤
│ 시간간 │ 일간⭐ │ 월간   │ 연간   │  ← 천간
│ 시간지 │ 일지   │ 월지   │ 연지   │  ← 지지
└────────┴────────┴────────┴────────┘
```

### 각 주(柱)의 의미

| 주 | 의미 | 나이대 | 관계 |
|:---:|:---|:---|:---|
| 연주 | 조상, 뿌리 | 0~15세 | 조부모, 외가 |
| 월주 | 부모, 성장환경 | 15~30세 | 부모, 형제 |
| 일주 | 본인, 배우자 | 30~45세 | 배우자 |
| 시주 | 자녀, 말년 | 45세~ | 자녀, 후배 |

### 일간 (日干)의 중요성

**일간은 "나 자신"을 나타내는 가장 중요한 글자입니다.**

- 모든 십성은 일간을 기준으로 계산
- 용신, 희신도 일간 기준
- 일간의 강약(신강/신약)이 해석의 핵심

## 7. 해석 시 주의사항

1. **일간 중심**: 모든 해석은 일간을 기준으로
2. **균형 확인**: 오행의 과다/부족 확인
3. **상생상극 정확히**: 목→화→토→금→수 (상생), 목→토→수→화→금 (상극)
4. **계절 고려**: 월지의 계절이 일간에 유리/불리한지
5. **지장간 활용**: 숨겨진 기운까지 분석

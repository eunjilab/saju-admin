# 합충형파해 해석 가이드

지지(地支)와 천간(天干) 사이의 관계를 분석하는 가이드입니다.

---

## 1. 천간합 (天干合) - 5합

두 천간이 만나 새로운 오행으로 변화합니다.

| 합 | 합화 오행 | 의미 |
|:---:|:---:|:---|
| 甲 + 己 | 土 | 중정지합 - 인의의 합, 가장 아름다운 합 |
| 乙 + 庚 | 金 | 인의지합 - 의리의 합, 강한 결속 |
| 丙 + 辛 | 水 | 위엄지합 - 권위의 합, 명예 지향 |
| 丁 + 壬 | 木 | 인수지합 - 음란의 합, 정(情) 강함 |
| 戊 + 癸 | 火 | 무정지합 - 무정한 합, 실리적 |

### 천간합 해석

**합이 되면:**
- 두 글자의 원래 기능이 약해짐
- 새로운 오행의 기운이 생김 (조건 충족 시)
- 관계: 합은 "끌림", "인연", "결합"

**합화 조건 (완전히 변하려면):**
- 월지가 합화 오행을 지지해야 함
- 다른 방해 요소가 없어야 함
- 조건 안 되면 "합이 되었으나 화(化)하지 않음"

### 합 vs 쟁합 (爭合)

```
쟁합: 하나의 글자를 두 글자가 동시에 합하려는 상황
예: 甲 - 己 - 甲 (두 甲이 하나의 己를 쟁합)
결과: 합의 효과가 약해지거나 무효화
의미: 삼각관계, 경쟁, 불안정
```

---

## 2. 육합 (六合) - 지지 합

인접한 지지끼리 합하여 새로운 오행이 됩니다.

| 합 | 합화 오행 | 계절 의미 |
|:---:|:---:|:---|
| 子 + 丑 | 土 | 겨울→봄 전환, 안정적 합 |
| 寅 + 亥 | 木 | 봄의 시작, 생명력 |
| 卯 + 戌 | 火 | 봄→가을, 열정적 합 |
| 辰 + 酉 | 金 | 가을 기운, 결실 |
| 巳 + 申 | 水 | 여름→가을, 지혜로운 합 |
| 午 + 未 | (土/火) | 한여름, 열정과 안정 |

### 육합 해석

**육합의 특징:**
- 가장 친밀한 관계
- 서로 끌어당김, 인연
- 해당 오행의 기운이 강해짐

**위치별 의미:**
| 위치 | 의미 |
|:---|:---|
| 연지-월지 합 | 조상/부모 덕, 좋은 가정환경 |
| 월지-일지 합 | 배우자복, 가정 화목 |
| 일지-시지 합 | 자녀복, 말년 안정 |

---

## 3. 삼합 (三合) - 3개 지지의 합

세 지지가 모여 하나의 오행 국(局)을 이룹니다.

| 삼합 | 생성 오행 | 구성 (생지-왕지-고지) |
|:---:|:---:|:---|
| 申子辰 | 水局 | 申(생) - 子(왕) - 辰(고) |
| 寅午戌 | 火局 | 寅(생) - 午(왕) - 戌(고) |
| 巳酉丑 | 金局 | 巳(생) - 酉(왕) - 丑(고) |
| 亥卯未 | 木局 | 亥(생) - 卯(왕) - 未(고) |

### 삼합 해석

**완전 삼합 (3개 모두 있음):**
- 해당 오행이 매우 강해짐
- 그 오행의 특성이 인생 전반에 영향
- 길신이면 대길, 흉신이면 주의

**반합 (半合) - 2개만 있음:**

| 반합 | 의미 |
|:---|:---|
| 생지 + 왕지 (예: 申子) | 생합 - 시작의 기운, 성장 중 |
| 왕지 + 고지 (예: 子辰) | 묘합 - 완성/결실 기운 |
| 생지 + 고지 (예: 申辰) | 공합 - 가장 약한 합, 중심 없음 |

---

## 4. 방합 (方合) - 계절 합

같은 계절의 3개 지지가 모입니다.

| 방합 | 오행 | 계절 | 구성 |
|:---:|:---:|:---:|:---|
| 寅卯辰 | 木 | 봄 | 동방합 |
| 巳午未 | 火 | 여름 | 남방합 |
| 申酉戌 | 金 | 가을 | 서방합 |
| 亥子丑 | 水 | 겨울 | 북방합 |

### 방합 해석

**특징:**
- 삼합보다 더 강한 오행 기운
- 계절의 기운이 집중됨
- 해당 오행이 압도적으로 강함

**주의:**
- 방합이 용신 오행이면 대길
- 방합이 기신 오행이면 그 기운이 너무 강해 문제

---

## 5. 충 (沖) - 정반대 충돌

180도 반대 지지끼리 충돌합니다.

| 충 | 특징 |
|:---:|:---|
| 子 ↔ 午 | 수화충 - 감정/열정 충돌, 심장/신장 주의 |
| 丑 ↔ 未 | 토토충 - 가치관 충돌, 소화기 주의 |
| 寅 ↔ 申 | 목금충 - 결단/추진 충돌, 간/폐 주의 |
| 卯 ↔ 酉 | 목금충 - 예술/실리 충돌, 가장 강한 충 |
| 辰 ↔ 戌 | 토토충 - 창고 충돌, 변화 암시 |
| 巳 ↔ 亥 | 화수충 - 지혜/열정 충돌, 순환기 주의 |

### 충의 해석

**충의 기본 의미:**
- 변동, 이동, 변화
- 충돌, 갈등, 분리
- 기존 것의 파괴와 새로운 시작

**위치별 충:**
| 위치 | 의미 |
|:---|:---|
| 연지-월지 충 | 조상/부모와 갈등, 객지 생활 |
| 월지-일지 충 | 부모/배우자 갈등, 가정 불안 |
| 일지-시지 충 | 배우자/자녀 갈등, 말년 불안 |
| 연지-일지 충 | 조상 덕 약함, 자수성가 |

**충의 양면성:**
- 나쁜 것을 충하면 → 흉이 물러남 (좋음)
- 좋은 것을 충하면 → 길이 물러남 (나쁨)
- 창고(辰戌丑未) 충 → 재물/기회 열림

---

## 6. 형 (刑) - 서로 해침

충보다 은밀하고 지속적인 해침입니다.

### 삼형 (三刑)

| 형 | 이름 | 구성 | 의미 |
|:---:|:---:|:---:|:---|
| 寅巳申 | 무은지형(無恩之刑) | 은혜를 배반 | 배신, 송사, 관재 |
| 丑戌未 | 무례지형(無禮之刑) | 예의 없음 | 형벌, 사고, 건강 |
| 子卯 | 무례지형(無禮之刑) | 무례한 관계 | 형벌, 예의 결여 |

### 자형 (自刑)

| 자형 | 의미 |
|:---:|:---|
| 辰 + 辰 | 스스로 해침, 자책 |
| 午 + 午 | 열정 과다, 번아웃 |
| 酉 + 酉 | 예민, 자해적 성향 |
| 亥 + 亥 | 생각 과다, 우울 |

### 형의 해석

**특징:**
- 충보다 은밀하고 느림
- 만성적 문제, 건강 이상
- 관재, 송사, 형벌 암시

---

## 7. 파 (破) - 깨뜨림

합을 방해하고 파괴합니다.

| 파 | 관계 |
|:---:|:---|
| 子 ↔ 酉 | 子丑합을 酉가 파괴 |
| 丑 ↔ 辰 | 丑子합을 辰이 파괴 |
| 寅 ↔ 亥 | 寅亥합이지만 파도 됨 |
| 卯 ↔ 午 | 卯戌합을 午가 파괴 |
| 巳 ↔ 申 | 巳申합이지만 파도 됨 |
| 未 ↔ 戌 | 戌卯합을 未가 파괴 |

### 파의 해석

**특징:**
- 합의 완성을 방해
- 좋은 일이 중간에 깨짐
- 계획 변경, 약속 파기

---

## 8. 해 (害/穿) - 해침

육합을 방해하는 관계입니다.

| 해 | 관계 | 의미 |
|:---:|:---|:---|
| 子 ↔ 未 | 子丑합을 未가 해침 | 정서적 상처 |
| 丑 ↔ 午 | 午未합을 丑이 해침 | 건강 문제 |
| 寅 ↔ 巳 | 寅亥합을 巳가 해침 | 송사, 구설 |
| 卯 ↔ 辰 | 辰酉합을 卯가 해침 | 은밀한 해침 |
| 申 ↔ 亥 | 亥寅합을 申이 해침 | 계획 차질 |
| 酉 ↔ 戌 | 戌卯합을 酉가 해침 | 배신, 이별 |

### 해의 해석

**특징:**
- 가장 은밀한 해침
- 가까운 사람에게서 오는 상처
- 육합 관계를 파괴

---

## 9. 합충형파해 우선순위

여러 관계가 겹칠 때 판단 순서:

```
1. 합 > 충 (합이 충을 무력화)
2. 충 > 형 (충이 더 강함)
3. 삼합 > 육합 (삼합이 더 강함)
4. 방합 > 삼합 (방합이 가장 강함)
```

### 합충동시 (合沖同時)

**합과 충이 동시에 있을 때:**
- 원칙: 합이 먼저, 충은 약해짐
- 단, 충하는 글자가 옆에 있으면 충 효과 발생

---

## 10. 해석 예시

### 예: 甲戌 / 丁卯 / 己酉 / 庚午

```
지지: 戌 - 卯 - 酉 - 午

발견된 관계:
1. 육합: 卯戌합(火)
2. 충: 卯酉충
3. 파: 卯午파

해석:
- 卯戌 육합이 있어 인연/결합 기운
- 그러나 卯酉충이 합을 방해 → 관계에 변동
- 卯午파까지 있어 완전한 합은 어려움
- 결론: 인연은 있으나 변화와 갈등 동반
```

# 대운/세운 해석 가이드

대운(大運)과 세운(歲運)은 시간에 따른 운의 흐름을 나타냅니다.

---

## 1. 대운 (大運) 기본 개념

### 대운이란?
- 10년 단위로 바뀌는 운의 큰 흐름
- 월주(月柱)를 기준으로 순행/역행
- 사주 원국과의 상호작용으로 길흉 판단

### 대운 방향 결정

| 성별 | 연간 양간 (甲丙戊庚壬) | 연간 음간 (乙丁己辛癸) |
|:---:|:---:|:---:|
| 남자 | 순행 ↗ | 역행 ↙ |
| 여자 | 역행 ↙ | 순행 ↗ |

**순행**: 월주 → 다음 간지로 진행 (예: 丁卯 → 戊辰 → 己巳)
**역행**: 월주 → 이전 간지로 진행 (예: 丁卯 → 丙寅 → 乙丑)

### 대운 시작 나이 계산

**생일부터 절입일까지의 날수 ÷ 3 = 대운 시작 나이**

- 순행: 생일 → 다음 절입일
- 역행: 생일 → 이전 절입일
- 3일 = 1년으로 환산

**예시:**
- 생일: 3월 15일
- 다음 절입일: 4월 5일 (청명)
- 날수: 21일 ÷ 3 = 7세부터 대운 시작

## 2. 대운 해석 방법

### 2-1. 대운과 원국의 관계

대운 천간/지지가 원국과 어떤 관계인지 확인합니다.

| 관계 | 의미 |
|:---|:---|
| 용신 운 | 가장 좋은 운, 발전/성취 |
| 희신 운 | 좋은 운, 도움이 됨 |
| 기신 운 | 나쁜 운, 어려움/장애 |
| 구신 운 | 매우 나쁜 운, 시련 |

### 2-2. 대운 천간/지지 분석

**대운 전반기 (1~5년)**: 천간 위주
**대운 후반기 (6~10년)**: 지지 위주

```
예: 대운 庚午 (30~39세)

30~34세: 庚 (천간) 영향 강함
- 일간과 庚의 십성 관계로 해석

35~39세: 午 (지지) 영향 강함
- 午의 지장간과 신살로 해석
```

### 2-3. 대운과 원국의 합충형파해

| 관계 | 해석 |
|:---|:---|
| 대운-원국 합 | 새로운 인연, 결합, 기회 |
| 대운-원국 충 | 변화, 이동, 전환점 |
| 대운-원국 형 | 건강/관재 주의, 은밀한 문제 |

## 3. 대운 주기별 의미

### 10대~20대 대운
- 학업, 진로 결정
- 부모 영향 강함
- 기반 형성 시기

### 30대~40대 대운
- 사회활동 전성기
- 결혼, 출산
- 직업적 성취

### 50대~60대 대운
- 성숙기, 결실
- 건강 관리 중요
- 자녀 독립

### 70대 이후 대운
- 노년기
- 여유와 정리
- 건강 최우선

## 4. 세운 (歲運) 기본 개념

### 세운이란?
- 1년 단위의 운
- 해당 연도의 천간지지
- 원국 + 대운 + 세운 = 종합 해석

### 2026년 세운
**丙午 (병오년)**
- 천간: 丙 (양화, 태양)
- 지지: 午 (양화, 한여름)
- 특징: 화(火) 기운 강함, 열정/활동적

---

## 5. 세운 해석 방법

### 5-1. 세운 천간 분석

세운 천간과 일간의 십성 관계를 봅니다.

```
예: 己土 일간, 2026년 丙午년

세운 천간 丙 → 己 기준 = 정인(正印)

해석:
- 정인의 해 = 학업, 도움, 보호의 기운
- 어머니/스승 인연 좋음
- 자격증/문서 행운
```

### 5-2. 세운 지지 분석

세운 지지의 지장간과 원국 지지와의 관계를 봅니다.

```
예: 己土 일간, 2026년 丙午년

세운 지지 午:
- 지장간: 己(비견), 丁(편인)
- 원국 지지와 합/충/형 확인

원국에 卯가 있다면:
- 卯午 = 파(破) → 계획 변동 주의
```

### 5-3. 세운과 대운의 조합

**세운 + 대운 조합이 중요합니다.**

| 조합 | 의미 |
|:---|:---|
| 대운 좋음 + 세운 좋음 | 최상의 해 |
| 대운 좋음 + 세운 나쁨 | 기복 있으나 버틸 수 있음 |
| 대운 나쁨 + 세운 좋음 | 일시적 행운, 근본 어려움 |
| 대운 나쁨 + 세운 나쁨 | 가장 어려운 해, 조심 |

## 6. 월운 (月運) 기본

### 월운이란?
- 1개월 단위의 운
- 해당 월의 간지
- 세운 내에서의 미세 변화

| 4월 | 청명~입하 | 壬辰 |

## 7. 대운/세운 종합 해석 예시

### 예: 己土 일간 (신정한 님 사례)

**원국:**
```
┌────────┬────────┬────────┬────────┐
│  시주  │  일주  │  월주  │  연주  │
├────────┼────────┼────────┼────────┤
│   庚   │   己   │   丁   │   甲   │
│   午   │   酉   │   卯   │   戌   │
└────────┴────────┴────────┴────────┘
```

**현재 대운:** 庚午 (24~33세)
- 천간 庚 = 상관 → 표현력, 재능 발휘
- 지지 午 = 편인 → 학업, 도움

**2026년 세운:** 丙午
- 천간 丙 = 정인 → 학업, 보호
- 지지 午 = 편인 → 인성 과다 주의

**종합 해석:**
```
대운 庚午 + 세운 丙午 조합

1. 인성(印星) 과다
   - 대운 午 + 세운 午 = 午午 (자형!)
   - 편인이 겹침 → 생각 많아짐, 게으름 주의

2. 상관(庚) + 정인(丙) 공존
   - 창의적 표현 + 학업 기회
   - 단, 결정 장애 주의

3. 卯酉충 영향
   - 원국 卯와 酉가 충
   - 세운 午가 卯를 파(破)
   - 변화/이동 가능성

4. 권장사항
   - 학업/자격증 도전 좋음
   - 행동력 유지 필요
   - 건강(심장/혈압) 주의
```

## 8. 대운/세운 주의사항

### 해석 시 주의할 점

1. **원국이 기본**: 대운/세운은 원국을 변화시키지 않음
2. **용신 확인**: 용신을 돕는 운인지, 해치는 운인지
3. **합충 확인**: 대운/세운이 원국과 합충형파해 확인
4. **복합 해석**: 천간+지지+신살 종합적으로
5. **시기 구분**: 대운 전반기/후반기 나누어 해석

### 좋은 대운의 특징
- 용신 오행이 들어옴
- 원국의 기신을 합거/충거
- 원국의 길신과 합
- 건록/정관/정인 등 길신 대운

### 어려운 대운의 특징
- 기신 오행이 강해짐
- 원국의 용신을 충/형
- 양인살/겁살 등 흉신 대운
- 공망이 많이 발생

---

## 9. 실전 해석 순서

```
1단계: 원국 분석
- 일간 강약 판단
- 용신/기신 확정

2단계: 대운 분석
- 대운 천간의 십성
- 대운 지지의 지장간/신살
- 원국과의 합충형파해

3단계: 세운 분석
- 세운 천간의 십성
- 세운 지지와 원국/대운 관계
- 대운+세운 조합 효과

4단계: 종합 판단
- 용신운인가 기신운인가
- 어떤 영역에 영향인가 (직업/재물/인연/건강)
- 구체적 조언 도출
```

# 대운/세운 해석 가이드

대운은 10년 단위, 세운은 1년 단위 운세예요.
원래 사주(원국)에 시간의 흐름이 더해지는 거예요.

---

## 1. 대운이 뭐예요?

**10년 동안의 큰 흐름**

예를 들어:
- 20대 대운이 좋으면 → 20대에 기회 많음
- 40대 대운이 힘들면 → 40대에 고비 있음

**대운은 월주에서 시작해서 순서대로 흘러가요**

---

## 2. 대운 방향 (순행/역행)

| 성별 | 연간이 양간 (갑병무경임) | 연간이 음간 (을정기신계) |
|:---:|:---:|:---:|
| 남자 | 순행 → | 역행 ← |
| 여자 | 역행 ← | 순행 → |

**순행:** 월주 다음 간지로 → 예) 정묘 → 무진 → 기사
**역행:** 월주 이전 간지로 ← 예) 정묘 → 병인 → 을축

---

## 3. 대운 해석하는 법

### 전반기 vs 후반기

**대운 10년을 5년씩 나눠요**

- **전반기 (1~5년):** 천간(위) 영향 강함
- **후반기 (6~10년):** 지지(아래) 영향 강함

예) 대운 庚午 (30~39세)
- 30~34세: 경(庚) 위주로 봐요
- 35~39세: 오(午) 위주로 봐요

### 대운이 좋다/나쁘다 판단

| 대운 종류 | 의미 |
|:---|:---|
| 용신 대운 | 최고! 원하는 거 이뤄짐 |
| 희신 대운 | 좋아요, 순조로움 |
| 기신 대운 | 힘들어요, 장애물 많음 |
| 구신 대운 | 매우 힘듦, 시련기 |

### 대운과 원국의 관계

| 관계 | 해석 |
|:---|:---|
| 대운이 원국과 합 | 새로운 인연, 기회 |
| 대운이 원국과 충 | 변화, 이동, 전환점 |
| 대운이 원국과 형 | 건강/법적 문제 주의 |

---

## 4. 나이대별 대운 의미

### 10대~20대 대운
- 학업, 진로 선택
- 부모 영향 강함
- 인생의 기초 쌓는 시기

**좋은 대운이면:** 공부 잘되고 좋은 기회
**힘든 대운이면:** 방황하거나 어려운 환경

### 30대~40대 대운
- 일 열심히 할 때
- 결혼, 출산
- 성과 나오는 시기

**좋은 대운이면:** 승진, 사업 성공
**힘든 대운이면:** 직장 스트레스, 가정 갈등

### 50대~60대 대운
- 인생 결실 맺는 때
- 건강 챙겨야 해요
- 자녀 독립

**좋은 대운이면:** 여유롭고 풍족
**힘든 대운이면:** 건강, 재물 조심

### 70대 이후 대운
- 노년기
- 건강이 최우선
- 편안한 마무리

---

## 5. 세운이 뭐예요?

**올해 1년의 운세**

예) 2026년 = 병오년
- 천간 병(丙) = 태양, 화
- 지지 오(午) = 한여름, 화

**2026년은 화(火) 기운이 강한 해예요**

---

## 6. 세운 해석하는 법

### 세운 천간 보기

세운 천간이 나한테 무슨 십성인지 봐요

예) 기토 일간, 2026년 병오년
- 병(丙)은 기토 기준 = 정인
- 올해 테마: 학업, 도움, 문서운

### 세운 지지 보기

세운 지지가 원국 지지랑 합/충하는지 봐요

예) 원국에 자(子)가 있는데 세운이 오(午)면
- 자오충! → 올해 변화 많음

### 대운 + 세운 조합

**둘 다 봐야 정확해요**

| 조합 | 의미 |
|:---|:---|
| 대운 좋음 + 세운 좋음 | 최고의 해! |
| 대운 좋음 + 세운 나쁨 | 힘들지만 버틸 수 있음 |
| 대운 나쁨 + 세운 좋음 | 반짝 행운, 근본 어려움 |
| 대운 나쁨 + 세운 나쁨 | 가장 조심해야 할 해 |

## 7. 월운 (1개월 운세)

세운 안에서 달마다 운세가 달라요

### 2026년 월별 운세 (참고)

| 월 | 간지 | 특징 |
|:---:|:---:|:---|
| 1월 | 기축 | 토 기운, 안정 |
| 2월 | 경인 | 금+목, 변화 시작 |
| 3월 | 신묘 | 금+목, 결정 필요 |
| 4월 | 임진 | 수+토, 지혜로운 선택 |
| 5월 | 계사 | 수+화, 감정 기복 |
| 6월 | 갑오 | 목+화, 활동적 |
| 7월 | 을미 | 목+토, 성장 |
| 8월 | 병신 | 화+금, 열정+결단 |
| 9월 | 정유 | 화+금, 섬세한 선택 |
| 10월 | 무술 | 토, 안정과 마무리 |
| 11월 | 기해 | 토+수, 준비 시기 |
| 12월 | 경자 | 금+수, 새해 준비 |

## 8. 실전 해석 예시

**예: 기토 일간, 현재 대운 경오(24~33세), 2026년 병오년**

```
대운 분석:
- 천간 경(庚) = 상관 → 표현력, 재능 발휘
- 지지 오(午) = 편인 → 학업, 도움

세운 분석:
- 천간 병(丙) = 정인 → 학업, 자격증 좋음
- 지지 오(午) = 편인 → 인성 겹침

조합 분석:
1. 대운 오(午) + 세운 오(午) = 오오 자형!
   → 열정 과다, 번아웃 주의

2. 인성(정인+편인) 과다
   → 생각 많아짐, 게으름 주의
   → 공부하기엔 좋은 해

3. 권장사항
   - 자격증, 공부 도전하기 좋아요
   - 생각만 하지 말고 행동하세요
   - 건강(심장, 혈압) 챙기세요
```

## 9. 좋은 대운 vs 힘든 대운

### 좋은 대운 특징
- 용신 오행이 들어옴
- 원국의 나쁜 글자를 충으로 없앰
- 원국의 좋은 글자와 합
- 건록, 정관, 정인 등이 오는 대운

### 힘든 대운 특징
- 기신 오행이 강해짐
- 원국의 용신을 충/형
- 양인살, 겁살 등 나쁜 신살 대운
- 공망이 많이 발생

---

## 10. 대운/세운 해석 순서

```
1단계: 원국 먼저
- 일간이 강한지 약한지
- 용신/기신 확인

2단계: 대운 분석
- 천간의 십성
- 지지의 신살
- 원국과 합/충 확인

3단계: 세운 분석
- 천간의 십성
- 지지와 원국/대운 관계

4단계: 종합 판단
- 용신운인지 기신운인지
- 어떤 분야에 영향인지 (직업/재물/연애/건강)
- 구체적 조언 도출
```

---

## 11. 해석 꿀팁

1. **원국이 기본** - 대운/세운은 원국을 바꾸지 않아요. 강화하거나 약화할 뿐
2. **10년 흐름 보기** - 세운 한 해보다 대운 10년이 더 중요해요
3. **전반기/후반기 나누기** - 같은 대운도 5년씩 다르게 봐야 해요
4. **충은 무조건 나쁜 게 아님** - 나쁜 걸 충하면 오히려 좋아요

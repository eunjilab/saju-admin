# 프리미엄 결과지 작성 가이드

프리미엄 패키지 (42~46페이지) 사주 결과지 작성을 위한 가이드입니다.

---

### 전체 페이지 구성 (42~46페이지)

| 페이지 | 내용 | 분량 |
|--------|------|------|

| 15-16 | 올해 운세 흐름 | 2페이지 |

## 3. 문체 및 톤

### 기본 원칙
- **친근한 존댓말**: "~예요", "~거예요", "~이기도 해요"
- **딱딱한 문체 금지**: "~입니다", "~습니다" ❌
- **긍정적 표현 우선**: 단점도 성장 가능성으로
- **구체적 예시 포함**: 추상적 표현 피하기
- **이모티콘**: 페이지당 1~2개만

### 어미 변주 (필수!)

| ❌ 피해야 할 어미 | ✅ 사용할 어미 |
|:---:|:---:|
| ~입니다 | ~이죠 / ~이기도 해요 |
| ~이에요 (반복) | ~예요 / ~거예요 / ~같아요 |
| ~해야 합니다 | ~해보세요 / ~하면 좋아요 |

### 톤 예시

**나쁜 예:**
> 己土는 토의 음간이며 곤토에 해당합니다. 사주에서 일간이 己인 경우 음토의 특성을 가집니다.

**좋은 예:**
> 당신은 己土 일간으로, 비옥한 논밭처럼 주변을 따뜻하게 품어주는 성품을 가지고 있어요. 포용력이 뛰어나 사람들이 자연스럽게 모여드는 매력이 있죠.

## 4. 섹션별 작성 가이드

### 4-1. 표지 (1페이지)

```
<!-- 페이지 1: 표지 -->

# 운명을 듣다 - OOO님의 사주 리포트

**프리미엄 패키지**

📅 [생년월일시]
🔮 [사주 4주 미리보기]
📌 생성일: 2026년 O월 O일

---

## 📋 올해 핵심 요약

1. **올해 테마**: [핵심 테마 1줄]
2. **핵심 키워드**: [키워드 3개]
3. **주의사항**: [주의점 1줄]

✨ **이 해의 대표 문장**
"[감정적 임팩트 문장 1줄]"
```

### 4-2. 사주 명식 + 구조 설명 (2-3페이지)

**포함 내용:**
1. 사주 명식표 (연주/월주/일주/시주)
2. 각 주별 십성
3. 오행 분포 그래프
4. "사주 한눈에 보기" 3줄 요약

**작성 예시:**
```
<!-- 페이지 2: 사주 명식 -->

## 당신의 사주 명식

┌────────┬────────┬────────┬────────┐
│  시주  │  일주  │  월주  │  연주  │
├────────┼────────┼────────┼────────┤
│   庚   │   己   │   丁   │   甲   │
│   午   │   酉   │   卯   │   戌   │
└────────┴────────┴────────┴────────┘

💡 **당신의 사주, 한눈에 보기**

• 己(기) = 비옥한 논밭 → 따뜻하게 품어주는 포용력
• 酉(유) = 가을 저녁 → 날카로운 직관과 심미안
• 己酉 조합 = "가을 논밭의 황금빛" → 현실적이면서도 예술적 감각

✨ 한마디로: "따뜻한 마음, 날카로운 눈"
```

### 4-3. 일간 해석 (4-5페이지)

**구성:**
1. 일간 설명 (본질)
2. 공감 포인트 4개
3. 성격 심화 분석

**작성 포인트:**
- 질문지 답변 직접 인용
- "~할 때 있죠?" 형식으로 공감
- 사주 근거 설명 (일간+십성+오행)

### 4-4. 오행 분포 + 양음 밸런스 (6-8페이지)

2. 강한 오행 해석
3. 약한 오행 해석
4. 양음 밸런스 분석
5. 오행 보완 조언

### 4-5. 성격 + 심리 패턴 (9-11페이지)

4. 타인이 보는 나 vs 내면의 나
5. 심리 패턴 분석

### 4-6. 신살 분석 (12-14페이지)

### 4-15. 이름의 기운 + 올해 방향 (35-36페이지)

**프리미엄 전용 - 역학 통합 섹션**

- 얼굴 특징과 운세

**공간 조언 (풍수):**
- 집/사무실 배치
- 행운의 방위

**용신 활용법:**

## 5. 3단계 해석 공식 (모든 섹션 적용)

| 단계 | 내용 | 예시 |
|:---:|:---|:---|
| ① 분석 | 명리적 사실 | "비겁이 강해 혼자 해결하려는 성향이에요." |
| ② 공감 | 감정 공감 | "그래서 도와주는 사람조차 피곤하게 느껴질 수 있죠." |
| ③ 제안 | 실천 조언 | "이럴 땐 '도움을 받는 연습'을 해보세요." |

- [ ] 십성 분포 정확히 반영했는가?
- [ ] 합충형파해 빠짐없이 확인했는가?
- [ ] 신살 정확히 해석했는가?
- [ ] 대운/세운 관계 분석했는가?
- [ ] 오행 과다/부족 확인했는가?
- [ ] META 블록 맨 앞에 있는가?
- [ ] 페이지 구분 형식 지켰는가?
- [ ] 건강운 참조 문구 포함했는가?
- [ ] 테마 색상 포함했는가?

### 흔한 실수 방지

| 실수 | 올바른 해석 |

## 7. 금기 사항

### 절대 하지 말 것

1. ❌ **수명/사망 언급**: "몇 살에 위험하다" 등
2. ❌ **불치병 암시**: "큰 병에 걸릴 수 있다" 등
3. ❌ **이혼/사별 단정**: "결혼이 안 된다" 등
4. ❌ **부정적 단정**: "재물운이 없다" 등
5. ❌ **미신적 표현**: "귀신이 붙었다" 등
6. ❌ **"소름" 단어 사용**: "공감 포인트"로 대체

### 대안적 표현

| 나쁜 표현 | 좋은 표현 |
|:---|:---|
| 재물운이 없다 | 스스로 만들어가는 재물 유형 |
| 결혼이 늦다 | 신중하게 선택하는 유형 |
| 건강이 나쁘다 | 이 부분을 관리하면 좋아요 |
| 흉한 운이다 | 조심하면서 성장하는 시기 |

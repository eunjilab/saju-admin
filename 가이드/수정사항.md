## 🔍 문제 분석

### 현재 스킬 파일에 있는 규칙
```
09_종합해석_원칙.md:
- "단일 요소로 단정짓지 마세요"
- "상충되면 둘 다 언급"
- 체크리스트 있음
```

### 그런데도 실수한 이유
**"피해야 할 것"을 쓸 때는 격국만 보고 단정지어 버림**

즉, "좋은 것" 쓸 때는 여러 요소 봤는데,
"나쁜 것/피할 것" 쓸 때는 체크를 안 한 거예요!

## 📝 수정 제안

### 1️⃣ 09_종합해석_원칙.md에 추가할 규칙

```markdown
## 🚨 "피해야 할 것" 작성 시 필수 체크 (NEW!)

**부정적 표현 쓰기 전에 반드시 확인:**

### ❌ 이렇게 쓰면 안 됨
- "군대 같은 조직 안 맞아요" (격국만 봄)
- "조직 생활 힘들어요" (십성만 봄)
- "~는 피하세요" (신살 확인 안 함)

### ✅ 쓰기 전 체크리스트
"피해야 할 환경/사람/직업" 쓰기 전에:

□ 양인살 있나? → 있으면 "군대/경찰 적합" 가능
□ 장성살 있나? → 있으면 "조직에서 성장" 가능
□ 반안살 있나? → 있으면 "안정적 환경 좋음" 가능
□ 협록 있나? → 있으면 "협업 시너지" 가능

**하나라도 해당되면 단정적 부정 표현 금지!**
→ "~도 있지만, ~도 있어서 상황에 따라 달라요"로 수정

### 📋 직업운 특별 규칙
| 격국이 | 근데 신살에 | 그러면 |
|:-------|:-----------|:-------|
| 상관격 (자유로움) | 양인살/장성살 있음 | "조직 안 맞음" ❌ 금지 |
| 비겁 과다 (독립) | 협록 있음 | "혼자가 좋음" ❌ 금지 |
| 화개살 (고독) | 장성살 있음 | "혼자 일해야 함" ❌ 금지 |
```

### 2️⃣ 프롬프트에 추가할 내용

```markdown
## ⚠️ 부정적 표현 작성 규칙 (필독!)

**"피해야 할 것" 쓰기 전 3초 멈추기!**

1. 격국/십성만 보고 "~안 맞아요" 쓰려고 했나?
2. 신살 목록에 반대되는 신살 있나?
   - 양인살 → 군대/결단력 조직 OK
   - 장성살 → 조직 성장 OK
   - 협록 → 협업 OK
   - 반안살 → 안정적 환경 OK
3. 있으면 → "둘 다 언급" 필수!

### 실수 방지 예시
❌ "상관격이라 군대식 조직 피하세요"
⬇️ (양인살 있는지 확인)
✅ "상관격이라 애매한 권위는 불편하지만, 양인살이 있어서 명확한 규칙의 조직에서는 오히려 빛나요"
```

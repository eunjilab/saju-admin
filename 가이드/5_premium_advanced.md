# N. 건강운 ✅ (프리미엄 전용)

> 오행 기반 체질 + 주의 부위 + 관리법. 의학적 진단 ❌ 경향성 안내 ✅

---

## 건강운 기본 구조

```
[체질 특성] 오행 기반 2~3문장
[이런 적 있죠?] 공감 포인트 2~3개
[주의할 부위] 장기/신체 부위
[이렇게 관리하면 좋아요] 구체적 행동
[올해 건강 포인트] 세운 연결
```

---

## ⚠️ 건강운 어투 규칙 (중요!)

```
절대 금지:
❌ "~병에 걸릴 수 있어요"
❌ "~가 안 좋아요"
❌ "~를 조심하세요" (공포 유발)

권장 어투:
✅ "~가 약해지기 쉬운 체질이에요"
✅ "~를 챙기면 더 건강해져요"
✅ "~에 신경 쓰면 좋아요"

필수 면책:
"이건 사주 기반 경향성이에요. 
정확한 건강 상태는 전문의 상담을 받으세요."
```

## 오행별 건강 체질

### 목(木) 기운 강하면

```
[체질 특성]
"목 기운이 강해요. 
에너지가 위로 올라가는 체질이에요.
활동적이고 성장 욕구가 강해요."

[이런 적 있죠?]
"화나면 머리가 아프거나 눈이 충혈된 적 있어요."
"스트레스 받으면 목이나 어깨가 뻣뻣해져요."
"술 마시면 다음 날 유독 힘들어요."

[주의할 부위]
✦ 간, 담낭 — 해독 기능
✦ 눈 — 피로, 충혈
✦ 근육, 힘줄 — 뻣뻣함, 경직

[이렇게 관리하면 좋아요]
✦ 술, 기름진 음식 줄이기
✦ 스트레스 받으면 바로 풀기 (운동, 산책)
✦ 눈 피로 관리 — 블루라이트 차단, 휴식
✦ 신맛 나는 음식 좋아요 (레몬, 매실)
✦ 초록색 채소 자주 먹기

[주의할 시기]
봄(2~4월)에 기운이 올라가서 두통, 혈압 주의
```

```
✦ 근육 — 쉽게 지침
✦ 손톱, 발톱 — 약해짐

[이렇게 관리하면 좋아요]
✦ 아침 10분 스트레칭으로 몸 깨우기
✦ 초록색 채소, 새싹 채소 챙기기

✦ 신맛 음식 (식초, 레몬)
```

## 올해 건강 포인트 (세운 연결)

```
[2026년 병오년 공통]
✦ 순수 화(火) 기운의 해 — 심장, 혈압, 혈액순환 특히 주의
✦ 상반기(1~3월): 불씨 점화기 → 에너지 상승, 열 관리 시작
✦ 여름(4~6월): 화기 정점 → 과열 주의, 심장·혈압 체크 필수
✦ 하반기(7~9월): 수확기 → 피로 누적 주의, 휴식 필요
✦ 겨울(10~12월): 재충전기 → 수면·휴식으로 회복

[올해 건강 꿀팁]
✦ 물 자주 마시기 — 열 식히기 (올해 최고의 해열제)
✦ 심호흡, 명상 — 심장 안정, 감정 조절
✦ 술, 카페인 줄이기 — 화기 과다 방지
✦ 냉탕, 수면 — 수(水)기운으로 균형
✦ 건강검진 꼭 받으세요 (심혈관 중심)
```

## 용신별 보완 조언 (생활 속 실천)

> 용신은 사주에서 가장 필요한 기운이에요.
> 일상에서 용신을 보완하면 에너지 밸런스가 좋아져요.

---

### 용신이 수(水)일 때

```
[본능적으로 이미 하고 있을 수 있어요]
"물을 많이 마시는 편이에요. 항상 물병 들고 다녀요."
"목욕이나 샤워 시간이 길어요. 물 소리가 좋아요."

[이렇게 보완하면 좋아요]
✦ 물 많이 드시죠? 잘하고 계신 거예요 👍 — 본능적으로 수 기운 채우는 중
✦ 거기에 수영이나 반신욕/족욕 더하면 더 좋아요
✦ 흐르는 물 소리 듣기 (빗소리, 계곡 영상)
✦ 색상: 블루, 블랙, 네이비 계열
✦ 방향: 북쪽 — 책상/침대 북향이면 좋아요
```

---

### 용신이 목(木)일 때

```
[본능적으로 이미 하고 있을 수 있어요]
"산책하면 기분이 좋아져요. 나무 많은 곳이 편해요."
"식물 키우는 거 좋아해요. 집에 화분 많아요."

[이렇게 보완하면 좋아요]
✦ 산책 좋아하시죠? 잘하고 계신 거예요 👍 — 목 기운 충전 중
✦ 아침 스트레칭으로 몸 깨우기
✦ 등산, 숲길 걷기 추천
✦ 새싹 채소, 초록색 음식 자주 먹기
✦ 색상: 초록색, 연두색 계열
✦ 방향: 동쪽 — 아침 햇살 받으면 좋아요
```

## O1. 성명학 — 이름의 기운 보완

### 개념
이름은 평생 불리는 "소리 에너지"입니다.
부족한 오행을 이름의 음운과 획수로 보완할 수 있어요.

### 자음별 오행 배속
| 오행 | 자음 | 의미 | 에너지 |
|------|------|------|--------|
| 木 | ㄱ, ㅋ | 성장, 시작 | 새싹이 뻗어나가는 힘 |
| 火 | ㄴ, ㄷ, ㄹ, ㅌ | 열정, 표현 | 불꽃처럼 퍼지는 힘 |
| 土 | ㅇ, ㅎ | 안정, 중심 | 대지처럼 받쳐주는 힘 |
| 金 | ㅅ, ㅈ, ㅊ | 결단, 정리 | 칼날처럼 끊는 힘 |
| 水 | ㅁ, ㅂ, ㅍ | 지혜, 유연 | 물처럼 흐르는 힘 |

### 해석 방법
```
1. 이름의 초성 추출
   예: "김민수" → ㄱ(木), ㅁ(水), ㅅ(金)

2. 사주 오행과 비교
   - 부족한 오행이 이름에 있으면 → "이름이 보완해주고 있어요"
   - 과다한 오행이 이름에도 있으면 → "에너지가 더 강해져요"

3. 결과지 표현 예시
   "이름 '민수'의 ㅁ(水)이 부족한 수 기운을 채워주고 있어요.
    유연함과 지혜의 에너지가 이름을 통해 보완됩니다."
```

### 진동 해석 (심화)
| 소리 위치 | 의미 | 해석 |
|----------|------|------|
| 첫소리 (초성) | 첫인상, 시작 에너지 | 사람들이 처음 느끼는 기운 |
| 중간소리 (중성) | 내면, 감정 에너지 | 속마음의 흐름 |
| 끝소리 (종성) | 마무리, 결과 에너지 | 일의 마무리 방식 |

### 결과지 출력 형식
```
## 💫 이름의 기운 (성명학)

당신의 이름 '{이름}'을 분석해봤어요.

✦ 초성 에너지: ㄱ(木) — 시작하는 힘이 강해요
✦ 전체 오행: 木 1개, 水 1개, 金 1개

[사주와 연결]
사주에서 부족한 화(火) 기운은 이름에 없어요.
→ 빨간색 소품이나 따뜻한 인사말로 보완하면 좋아요.

[이름 에너지 정리]
이름이 가진 金 기운이 결단력을 더해주고,
水 기운이 유연함을 보완해줍니다.
```

[출력 예시]
```
"이름 '{이름}'의 초성을 살펴봤어요.

{이름}이라는 이름에는 {오행1}의 {특성1}과 {오행2}의 {특성2}이 담겨 있어요.
부족한 {용신}을 {보완/강화}해주는 조합이에요.

```

## O2. 주역 — 방향과 선택의 지혜

### 📦 패키지별 주역 해석 깊이
| 패키지 | 주역 해석 방식 |
|--------|---------------|
| 라이트 | ❌ 주역 해석 없음 |
| 스탠다드 | ⚪ 힌트형 — 괘象 키워드만 간략히 언급 (1~2문장) |
| 프리미엄 | ✅ 전체 — 괘象 + ② 심리 패턴 + ④ 천문 연동 |

**스탠다드 힌트형 예시:**
```
💡 주역 힌트: 올해는 離괘(리괘)의 기운이에요.
✦ 드러내고 표현하는 게 길한 해. 숨기지 말고 보여주세요.
```

**프리미엄 전체 예시:**
```
🔮 올해의 방향 — 離괘(리괘)
불의 기운, 밝음과 표현의 시기예요.

✦ 괘象 의미: 안에 있던 것을 밖으로 드러내는 때
✦ 방향: 숨기지 말고 보여주세요. PR, 자기표현, 도전에 유리해요.

💡 ② 심리 패턴:
離괘 시기에는 "인정받고 싶은 욕구"가 강해져요.
✦ 좋은 점: 동기부여 ↑, 행동력 ↑
⚠️ 주의점: 조급함 주의, 결과에 집착하면 지쳐요

💡 ④ 천문 연동:
✦ 입하(5/5) ~ 하지(6/21): 離괘 기운 최고조
✦ 이 시기에 시작한 일이 가장 잘 풀려요
```

### 개념
주역(易學)은 변화의 원리를 64괘로 설명합니다.
사주가 "타고난 기운"이라면, 주역은 "지금 흐름의 방향"을 알려줘요.

### 올해 운세와 연결하는 방법
```
1. 세운(올해 천간지지)의 성격 파악
2. 일간과의 관계에서 괘象 유추
3. 현재 시기에 맞는 방향 제시
```

### 기본 괘象 해석 (자주 쓰는 8가지)
| 괘 | 이름 | 의미 | 방향 조언 |
|----|------|------|----------|
| ☰ | 乾(건) | 하늘, 강건 | 적극적으로 추진해도 좋은 시기 |
| ☷ | 坤(곤) | 땅, 수용 | 기다리며 내실을 다지는 시기 |
| ☵ | 坎(감) | 물, 위험 | 신중하게, 무리하지 않는 시기 |
| ☲ | 離(리) | 불, 밝음 | 드러내고 표현하는 시기 |
| ☳ | 震(진) | 우레, 움직임 | 새롭게 시작하는 시기 |
| ☴ | 巽(손) | 바람, 유연 | 순응하며 기회 잡는 시기 |
| ☶ | 艮(간) | 산, 멈춤 | 잠시 멈추고 정리하는 시기 |
| ☱ | 兌(태) | 연못, 기쁨 | 사람들과 교류하는 시기 |

### 세운별 괘象 매칭 예시
```
[2025년 乙巳년 → 離(리)괘 성격]
→ 드러남, 밝아짐, 표현의 해
→ "숨겨왔던 것을 꺼내놓기 좋은 시기"

[2026년 丙午년 → 鼎(정)괘 성격]
→ 완성, 변혁, 재탄생의 해
→ "불이 솥에 담겨 완성되는 시기"

[2027년 丁未년 → 坤(곤)괘 성격]
→ 수용, 안정, 내실의 해
→ "기다리며 내실을 다지는 시기"
```

### 심리 패턴 연결
| 괘 성격 | 심리 경향 | 조언 |
|--------|----------|------|
| 乾형 | 추진력 강함, 완벽주의 | 쉬어가는 것도 전략이에요 |
| 坤형 | 수용적, 기다림 | 때가 되면 움직여야 해요 |
| 坎형 | 신중함, 걱정 많음 | 너무 움츠리지 마세요 |
| 離형 | 표현 욕구, 주목받고 싶음 | 진정성이 빛을 발해요 |

## O3. 택일학 — 실행의 타이밍

### 개념
택일(擇日)은 "좋은 날을 고르는" 학문입니다.
사주와 운의 흐름에 맞는 시점을 찾아 실행 효과를 높여요.

### 월별 기운과 추천 행동
| 월 | 절기 | 왕성한 오행 | 추천 행동 |
|----|------|------------|----------|
| 1~2월 | 소한~우수 | 水 | 계획, 공부, 내면 정리 |
| 3~4월 | 경칩~곡우 | 木 | 시작, 창업, 새 프로젝트 |
| 5~6월 | 입하~하지 | 火 | 표현, 발표, 사람 만남 |
| 7~8월 | 소서~처서 | 土 | 안정, 정리, 기반 다지기 |
| 9~10월 | 백로~상강 | 金 | 결단, 계약, 마무리 |
| 11~12월 | 입동~동지 | 水 | 휴식, 준비, 에너지 충전 |

### 사주 오행과 택일 연결
```
[용신이 火인 사람]
→ 5~6월(火旺月)에 중요한 일 추진하면 좋아요
→ 11~12월(水旺月)은 무리하지 말고 준비 시기로

[용신이 金인 사람]
→ 9~10월(金旺月)에 계약, 결정하면 좋아요
→ 3~4월(木旺月)은 새 시작보다 기존 일 마무리
```

### 특별한 날 추천
| 상황 | 추천 시기 | 이유 |
|------|----------|------|
| 창업/시작 | 입춘 후 ~ 청명 전 | 木 기운 상승, 새 시작 에너지 |
| 계약/결정 | 백로 후 ~ 한로 전 | 金 기운 상승, 결단의 시기 |
| 고백/프로포즈 | 입하 후 ~ 망종 전 | 火 기운 상승, 표현의 시기 |
| 이사/이동 | 용신 오행 왕성월 | 개인 사주에 맞춤 |
| 휴식/충전 | 동지 전후 2주 | 水 기운 정점, 내면 회복 |

### 2026년 주요 길일 예시
```
[입춘] 2월 4일 — 새해 에너지 시작, 계획 세우기 좋은 날
[춘분] 3월 20일 — 균형의 날, 결정하기 좋은 날
[입하] 5월 5일 — 활동 에너지 시작, 실행하기 좋은 날
[하지] 6월 21일 — 양 에너지 정점, 드러내기 좋은 날
[입추] 8월 7일 — 수확 준비, 마무리 시작
[추분] 9월 22일 — 균형의 날, 계약하기 좋은 날
```

### 결과지 출력 형식


## O4. 관상학 — 외형의 기운 조율

### 📦 패키지별 감각/표현 해석 깊이
| 패키지 | 감각/표현 해석 방식 |
|--------|-------------------|
| 라이트 | ⚪ 일부 — 오행 색상 1~2개만 간략 언급 |
| 스탠다드 | ✅ 색·온도감 조언 — 부족 오행 보완 색상 + 계절감 |
| 프리미엄 | ✅ 오감 기반 완전 반영 — ⑤ 시각/촉각/후각/공간 전체 |

**라이트 예시:**
```
💡 금(金) 부족 → 흰색, 회색 포인트 추천
```

**스탠다드 예시:**
```
🎨 외형 조언
✦ 금(金) 부족 → 흰색, 회색 포인트로 결단력 보완
✦ 따뜻한 계절(여름): 차가운 톤 추가로 균형
✦ 깔끔한 라인의 옷이 당신의 신뢰감을 높여줘요
```

**프리미엄 예시:**
```

💡 시각 (색감):
✦ 메인 컬러: 흰색, 회색 — 결단력·추진력 강화
✦ 포인트 컬러: 금색 — 중요한 날 자신감 부스터
✦ 피해야 할 색: 과한 빨강 — 화(火) 과다 시 감정 폭발

💡 촉각 (소재):
✦ 추천: 깔끔한 면, 린넨 — 정돈된 기운
✦ 피하기: 너무 부드러운 소재 — 결단력 약화

💡 후각 (향):
✦ 추천: 시트러스, 민트 계열 — 명확한 사고
✦ 피하기: 너무 달콤한 향 — 집중력 분산

🌡️ 계절별 조절:
✦ 여름(火旺): 차가운 색 추가, 시원한 소재
✦ 겨울(水旺): 따뜻한 포인트, 두꺼운 질감
```

### 개념
관상(觀相)은 얼굴과 외형에 나타난 기운을 읽는 학문입니다.
사주의 오행 불균형을 색감, 스타일, 표정으로 보완할 수 있어요.

### 오행별 외형 특징
| 오행 | 얼굴 특징 | 체형 경향 | 인상 |
|------|----------|----------|------|
| 木 | 긴 얼굴, 높은 이마 | 키 크고 마른 편 | 지적, 성장하는 느낌 |
| 火 | 뾰족한 턱, 밝은 눈 | 상체 발달 | 열정적, 활발한 느낌 |
| 土 | 넓은 얼굴, 두꺼운 입술 | 탄탄한 체형 | 안정감, 믿음직한 느낌 |
| 金 | 각진 얼굴, 날카로운 눈매 | 균형 잡힌 체형 | 단정함, 날카로운 느낌 |
| 水 | 둥근 얼굴, 부드러운 눈 | 하체 발달 | 유연함, 부드러운 느낌 |

### 부족한 오행 보완법 (색감/스타일)
| 부족 오행 | 보완 색상 | 스타일 조언 |
|----------|----------|------------|
| 木 부족 | 초록, 청록 | 세로 줄무늬, 자연스러운 소재 |
| 火 부족 | 빨강, 주황, 보라 | 화려한 포인트, 밝은 메이크업 |
| 土 부족 | 노랑, 베이지, 브라운 | 편안한 핏, 자연스러운 톤 |
| 金 부족 | 흰색, 회색, 금색 | 깔끔한 라인, 정돈된 스타일 |
| 水 부족 | 검정, 파랑, 남색 | 유연한 소재, 물결 패턴 |

### 표정과 기운
```
[웃는 표정] → 火 기운 상승
→ 화(火) 부족한 사람에게 추천
→ "자주 웃으면 운이 트여요"

[차분한 표정] → 水 기운 상승
→ 수(水) 부족한 사람에게 추천
→ "급하게 굴지 않으면 운이 따라와요"

[당당한 표정] → 金 기운 상승
→ 금(金) 부족한 사람에게 추천
→ "자신감 있는 표정이 기회를 부릅니다"
```

## O5. 풍수학 — 공간의 기운 조율

### 개념
풍수(風水)는 공간의 에너지 흐름을 읽고 조율하는 학문입니다.
사주의 부족한 오행을 공간 배치, 방위, 소품으로 보완할 수 있어요.

### 방위별 오행
| 방위 | 오행 | 의미 | 강화 효과 |
|------|------|------|----------|
| 동쪽 | 木 | 성장, 시작 | 새로운 일, 건강 |
| 남쪽 | 火 | 명예, 표현 | 인기, 성공 |
| 중앙 | 土 | 안정, 균형 | 건강, 가정 |
| 서쪽 | 金 | 결실, 재물 | 수입, 저축 |
| 북쪽 | 水 | 지혜, 유연 | 공부, 아이디어 |

### 부족한 오행 보완법 (공간)
| 부족 오행 | 방위 | 소품/인테리어 |
|----------|------|--------------|
| 木 부족 | 동쪽 활용 | 초록 식물, 나무 가구, 세로형 소품 |
| 火 부족 | 남쪽 활용 | 조명 밝게, 빨간 소품, 촛불 |
| 土 부족 | 중앙 활용 | 도자기, 황토색 소품, 네모난 형태 |
| 金 부족 | 서쪽 활용 | 금속 소품, 흰색 인테리어, 원형 |
| 水 부족 | 북쪽 활용 | 작은 분수, 어항, 물결 패턴 |

### 공간별 조언
```
[침실]
→ 머리 방향을 용신 오행 방위로
→ 예: 화(火) 용신 → 남쪽으로 머리 두고 잠자기

[책상/작업공간]
→ 앉았을 때 용신 방위를 바라보게
→ 예: 금(金) 용신 → 서쪽을 바라보며 일하기

[거실]
→ 부족한 오행 소품 배치
→ 예: 목(木) 부족 → 동쪽에 큰 화분 배치
```

### 간단 풍수 체크리스트
```
✅ 현관: 밝고 깨끗하게 (기운이 들어오는 곳)
✅ 침실: 거울이 침대 비추지 않게
✅ 책상: 문을 등지지 않게 배치
✅ 거실: 소파가 현관을 바라보게
✅ 화장실: 문 항상 닫아두기
```

### 결과지 출력 형식
```

✦ 중요 활동: 동쪽에서 하면 좋아요
✦ 책상 위치: 동쪽 벽면 추천
✦ 침대 방향: 동쪽으로 머리 두기

[소품 추천]
✦ 동쪽에 큰 화분이나 나무 소품
✦ 초록색 쿠션이나 커튼
✦ 세로로 긴 형태의 장식품

```

# 프리미엄 결과지 해석 가이드

프리미엄 패키지 사주 해석 시 참조하는 **해석 방법** 가이드입니다.
(페이지 구조, 문체, META 블록 등 형식은 프롬프트를 따르세요)

---

## 0. 핵심 원칙 (모든 해석에 적용!)

### 4가지 필수 요소
1. **직접적 단어** - 돌려 말하지 않기
2. **자연스러운 문체** - AI 같지 않게, 대화하듯이
3. **사주 근거** - 왜 그런지 명확히
4. **실용적 해결법** - 구체적으로 뭘 해야 하는지

### ❌ 나쁜 예시 (두리뭉실)
> "건강에 주의가 필요한 시기예요."
> "이 시기는 조심하면 좋겠어요."
> "관계에 신경 쓰면 좋을 것 같아요."

### ✅ 좋은 예시 (직접적)
> "8월에 심장이랑 혈압 조심하세요. 화(火) 기운이 과해져서 그래요. 야근 줄이고, 매운 음식 피하고, 물 많이 드세요."
> "6월에 돈 나갈 일 생겨요. 비겁이 강해지는 달이라 그래요. 보증 서지 마세요. 친구 돈 빌려달라고 해도 거절하세요."
> "시어머니랑 트러블 생길 수 있어요. 원진살 때문이에요. 이 달은 명절 방문 짧게 하세요."

## 1. 섹션별 해석 포인트

### 1-1. 일간 해석
**참조:** [saju-basics.md], [sipsung-guide.md]

- 일간의 자연물 비유로 설명
- 핵심 성향 3-4가지 도출
- 고객 고민과 연결하여 "왜 그런지" 설명

### 1-2. 오행 분포 분석
**참조:** [saju-basics.md]

- 강한 오행 → 과잉 시 나타나는 특성
- 약한 오행 → 부족 시 나타나는 특성
- 보완 방법 (색상, 방향, 계절 등)

### 1-3. 성격 + 심리 패턴
**참조:** [sipsung-guide.md], [byeongjon-guide.md]

- 십성 분포로 성격 도출
- 가장 강한 십성 중심으로 해석
- 병존 구조 있으면 특수 해석 추가

### 1-4. 신살 분석
**참조:** [sinsal-guide.md]

- 길신: 어떤 복이 있는지 구체적으로
- 양면신: 좋게 쓰는 방법 / 주의점
- 흉신: **뭐가 문제고, 어떻게 피하는지 직접 말하기**

### 1-5. 올해 운세 + 월별 운세
**참조:** [daeun-guide.md]

- 세운과 원국 관계 분석
- 용신운인지 기신운인지 판단
- 좋은 달: 뭘 하면 좋은지 구체적으로
- 안 좋은 달: 뭘 피해야 하는지 구체적으로

### 1-6. 연애/결혼운
**참조:** [sipsung-guide.md], [hapchung-guide.md]

- 재성(남)/관성(여) 분석
- 일지(배우자궁) 분석
- 합충 관계로 인연 흐름 설명

### 1-7. 직업/진로운
**참조:** [sipsung-guide.md], [gyeokguk-guide.md]

- 격국으로 적합 직업군 도출
- 십성 분포로 직업 스타일 분석
- 용신 활용 직업 추천

#### 🔴 직업운 "피해야 할 것" 작성 전 필수 교차검증!

**"피해야 할 직업/환경" 섹션 쓰기 전:**

| 확인 | 신살 | 있으면 |
|:----:|:----:|:-------|
| □ | 양인살 | "군대/경찰/외과의사 적합" 언급 |
| □ | 장성살 | "조직에서 성장" 언급 |
| □ | 협록 | "협업 시너지" 언급 |
| □ | 반안살 | "안정적 환경 OK" 언급 |

**하나라도 있으면:**
→ "~는 안 맞아요" 대신
→ "~는 불편할 수 있지만, ~도 있어서 ~한 환경에서는 오히려 좋아요"

### 1-8. 재물/사업운
**참조:** [sipsung-guide.md]

- 재성 유무 및 강약 분석
- 식상생재 흐름 확인
- 재물 패턴 (벌이/지출/저축)

### 1-9. 건강운
**참조:** [saju-basics.md]

**오행-건강 연결:**
| 오행 | 장기 | 과다 시 | 부족 시 |
|:---:|:---|:---|:---|
| 木 | 간/담/눈 | 화병, 분노 | 피로, 소화불량 |
| 火 | 심장/소장 | 혈압, 염증 | 순환 문제 |
| 土 | 비장/위장 | 당뇨, 비만 | 소화불량 |
| 金 | 폐/대장/피부 | 호흡기 | 면역력 저하 |
| 水 | 신장/방광 | 부종 | 생식기, 뼈 |

### 1-10. 대운 흐름
**참조:** [daeun-guide.md]

- 현재 대운 상세 분석
- 향후 10년 대운 미리보기
- 대운 전환 시기 주의점

### 1-11. 귀인 / 피할 사람
**참조:** [sinsal-guide.md]

- 천을귀인, 천덕귀인 등으로 도움 되는 띠
- 충, 형, 해 관계로 주의할 띠
- 피할 사람: 왜 안 맞는지 + 어떻게 대처하는지

### 1-12. 인연상 + 배우자상
- 일지 육합으로 인연 유형
- 일간 오행으로 배우자 외모/성격
- 만남 시기/장소 예측

### 1-13. 용신과 번아웃
**참조:** [gyeokguk-guide.md]

- 용신 상세 설명 및 활용법
- 기신 과다 시 번아웃 패턴
- 에너지 회복 방법

## 2. 위험/주의 시기 표현법

### 원칙
**직접 말하기 + 근거 + 구체적 해결법**

### 공식
```
[언제] + [뭐가 문제인지 직접] + [왜: 사주 근거] + [어떻게 피하는지 구체적으로]
```

### 예시

**건강:**
> "8월에 심장이랑 혈압 조심하세요. 화(火) 기운이 과해져서 그래요. 이 달은:
> - 야근 줄이세요
> - 매운 음식, 술 피하세요
> - 물 하루 2리터 드세요
> - 저녁 9시 전에 주무세요"

**사고:**
> "3월에 교통사고 조심하세요. 역마살이 충을 맞는 달이에요.
> - 장거리 운전 피하세요
> - 급한 약속 잡지 마세요
> - 꼭 가야 하면 대중교통 이용하세요"

**돈:**
> "6월에 돈 크게 나가요. 비겁운이라 그래요.
> - 보증 절대 서지 마세요
> - 투자 권유 다 거절하세요
> - 친구가 돈 빌려달라고 해도 안 된다고 하세요"

**관계:**
> "9월에 시어머니/장모님이랑 트러블 생겨요. 원진살 때문이에요.
> - 명절 방문 하루만 하세요
> - 민감한 얘기(돈, 육아) 피하세요
> - 남편/와이프가 중간에서 말하게 하세요"

**직장:**
> "4월에 상사랑 충돌해요. 관살혼잡 시기라 그래요.
> - 의견 충돌 나면 한 발 빼세요
> - 이 달은 보고서 2번 검토하세요
> - 회식 자리에서 술 적게 드세요"

### 5-1. HTML 주석 태그 시스템

각 섹션 내용을 **감정 흐름**에 따라 태그로 구분하세요.

| 태그 | 용도 | 예시 |
|:-----|:-----|:-----|
| `<!-- 공감 -->` | 독자 감정에 공감 | "이런 적 있죠?" |
| `<!-- 왜그런지 -->` | 명리학적 근거 설명 | "그게 비겁이 강해서 그래요" |
| `<!-- 주의 -->` | 주의해야 할 점 | "근데 이건 조심해야 해요" |
| `<!-- 희망 -->` | 희망적인 메시지 | "그래도 이런 장점이 있어요" |
| `<!-- 조언 -->` | 실질적 해결법 | "이렇게 해보세요" |

**사용 예시:**
```markdown
<!-- 공감 -->
"밖에서는 조용한데, 친한 사람 앞에서는 말이 엄청 많아지죠?"

맞아요. 丁 일간의 전형적인 특징이에요.
처음 보는 사람 앞에선 관찰 모드. 편한 사람 앞에선 수다쟁이 모드.
<!-- /공감 -->

<!-- 왜그런지 -->
왜 그런지 사주로 볼게요.

도대성님은 丁 일간에 **비겁이 4개**나 돼요.
자기 에너지가 매우 강해서 남에게 휘둘리지 않아요.
<!-- /왜그런지 -->
```

---

### 5-2. 문체 규칙

**종결어미:**
- ✅ "~예요", "~거든요", "~있죠?", "~거예요"
- ❌ "~합니다", "~입니다" (딱딱함)

**공감 유도 패턴:**
- "이런 적 있죠?" → "맞아요. 그게 바로~"
- "혹시 이런 적 있으세요?"
- "~한 적 있을 거예요"

**피해야 할 표현:**
| ❌ 피하기 | ✅ 대신 사용 |
|:----------|:-------------|
| "소름 돋는" | "공감 포인트" |
| "~할 것으로 예상됩니다" | "~할 수 있어요" |
| "~하시기 바랍니다" | "~해보세요" |
| "~인 것 같습니다" | "~거든요", "~예요" |

### 5-3. 공감 포인트 4개 형식

각 주요 섹션(일간 해석, 성격 분석 등)에 **공감 포인트 4개**를 넣으세요.

**형식:**
```markdown
### 💫 공감 포인트

<!-- 공감 -->
**공감 포인트 1**
"밖에서는 조용한데, 친한 사람 앞에서는 말이 엄청 많아지죠?"

맞아요. 丁 일간의 전형적인 특징이에요.
처음 보는 사람 앞에선 관찰 모드. 편한 사람 앞에선 수다쟁이 모드.
"얘가 원래 이렇게 말 많았어?" 이런 소리 들어본 적 있을 거예요.

---

**공감 포인트 2**
"한번 빠지면 진짜 깊이 빠지죠? 덕질이든, 취미든, 사람이든."

丁 일간은 관심이 생기면 올인하는 스타일이에요.
대충 좋아하는 게 없어요. 좋아하면 파고들고, 아니면 관심 자체가 없어요.

---

**공감 포인트 3**
"티 안 내도 알아봐 주는 사람 있으면, 그때 진짜 감동받죠?"

도대성님은 감정을 잘 드러내지 않는 편이에요.
근데 속으론 인정받고 싶은 마음이 분명히 있어요.

---

**공감 포인트 4**
"참고 참다가 한번 터지면 본인도 놀랄 만큼 터지죠?"

이것도 丁 일간의 특징이에요.
평소엔 "괜찮아요" 하다가, 임계점 넘으면 폭발해요.
<!-- /공감 -->
```

### 5-4. 오행 막대그래프 (텍스트 아트)

오행 분포는 **텍스트 막대그래프**로 시각화하세요.

**형식:**
```
화(火) ████████████████████████ 5.9개 ⭐ 최강
목(木) ██████████ 2.6개
수(水) ████ 1개
토(土) ███ 0.9개
금(金) █ 0.3개 ⚠️ 최약
```

**막대 길이 기준:**
- 1개 = █████ (5칸)
- 0.5개 = ██ (2칸)
- 최강 = ⭐ 표시
- 최약 (1 미만) = ⚠️ 표시

---

### 5-5. 감정 흐름 순서

각 섹션은 아래 순서를 따르세요:

```
1. 공감 (<!-- 공감 -->)
   → "이런 적 있죠?" 질문으로 시작

2. 왜그런지 (<!-- 왜그런지 -->)
   → 사주 근거로 설명

3. 주의점 (<!-- 주의 -->)
   → 조심해야 할 것

4. 희망 (<!-- 희망 -->)
   → 긍정적 메시지

5. 조언 (<!-- 조언 -->)
   → 실질적 해결법
```

**전체 흐름:**
공감으로 마음을 열고 → 분석으로 이해시키고 → 주의점으로 경각심 주고 → 희망으로 용기 주고 → 조언으로 행동 유도

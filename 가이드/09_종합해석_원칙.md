# 종합 해석 원칙 (필독!)

사주 해석 시 반드시 지켜야 할 원칙들이에요.

---

## ⚠️ 핵심: 단일 요소로 단정짓지 마세요!

### 해석 순서
1. 일간 + 신강/신약 + 용신/기신 (기본 뼈대)
2. 십성 분포 (어떤 에너지가 강한가)
3. 오행 분포 (넘치고 부족한 것)
4. 격국 (큰 틀의 성향)
5. 신살 (특수한 기운)
6. **⭐ 상충점 확인 → 둘 다 언급!**

---

## 🚨 1. 데이터 우선순위

### 절대 규칙
```
프롬프트에 있는 데이터 > 스킬 파일의 공식/이론
```

| 항목 | 사용할 데이터 | ❌ 하면 안 되는 것 |
|:-----|:-------------|:-----------------|
| 월운 | 프롬프트의 월운 표 그대로 | "1월=寅월" 임의 공식 적용 |
| 신살 | 프롬프트의 계산기 결과만 | 스킬 파일 보고 직접 계산 |
| 십성 | 프롬프트의 십성 분포 | 직접 계산해서 다른 값 사용 |
| 용신 | 프롬프트의 용신/기신 | 임의로 다른 용신 판단 |

### 왜?
- 프롬프트의 데이터 = 검증된 계산기 결과
- 스킬 파일 = 해석 방법/톤앤매너 참고용
- 직접 계산하면 틀릴 수 있음!

## 🚨 2. 종합 해석 원칙

### 원칙
서로 다른 요소가 반대 방향을 가리키면, **하나를 선택하지 말고 둘 다 설명**

### 예시

| 상황 | ❌ 잘못된 해석 | ✅ 올바른 해석 |
|:-----|:--------------|:--------------|
| 상관격 + 양인살 | "상관격이라 조직 안 맞아요" | "창의적 표현을 원하지만, 양인살이 있어서 명확한 규칙의 조직에선 오히려 빛나요" |
| 비겁 과다 + 협록 | "혼자 하는 게 좋아요" | "독립심이 강하지만, 협록이 있어서 협업할 때 시너지도 나요" |
| 화개살 + 장성살 | "혼자 예술하는 게 맞아요" | "예술적 감각이 있으면서, 조직에서 성장하는 힘도 있어요" |

## 📋 3. 카테고리별 해석 체크리스트

### 모든 카테고리 공통 순서

```
1단계: 일간 특성 확인
2단계: 십성 분포 확인 (뭐가 강한가)
3단계: 오행 분포 확인 (넘치고 부족한 것)
4단계: 격국 확인 (큰 틀의 성향)
5단계: 신살 확인 (특수한 기운)
6단계: ⭐ 상충점 확인 → 있으면 둘 다 언급!
```

---

### 직업운 체크리스트

| 확인 항목 | 보는 것 | 예시 |
|:----------|:--------|:-----|
| □ 격국 | 큰 성향 | 상관격 → 창의적, 표현 |
| □ 십성 분포 | 강한 에너지 | 비겁 과다 → 경쟁, 독립 |
| □ 신살 | 직업 관련 | 양인살 → 군인/경찰, 천의성 → 의료 |
| □ 용신 | 필요한 환경 | 금 용신 → 체계적 환경 |
| □ **상충점** | 서로 다른 방향 | 격국 vs 신살 다르면 → 둘 다 언급! |

**상충 해석 예시:**
- 상관격(자유) + 양인살(조직) → "창의적이면서도 명확한 규칙 있는 조직에서 빛남"
- 화개살(예술/고독) + 장성살(조직 성장) → "예술적 감각 + 조직에서 인정받는 힘 둘 다 있음"

---

### 🚨 "피해야 할 것" 작성 전 필수 체크!

**부정적 표현 쓰기 전에 반드시 확인:**

#### ❌ 이렇게 쓰면 안 됨
- "군대 같은 조직 안 맞아요" (격국만 봄)
- "조직 생활 힘들어요" (십성만 봄)
- "~는 피하세요" (신살 확인 안 함)

#### ✅ 쓰기 전 체크리스트
"피해야 할 환경/사람/직업" 쓰기 전에:

| 확인 | 신살 | 있으면 |
|:----:|:----:|:-------|
| □ | 양인살 | "군대/경찰 적합" 가능 |
| □ | 장성살 | "조직에서 성장" 가능 |
| □ | 반안살 | "안정적 환경 좋음" 가능 |
| □ | 협록 | "협업 시너지" 가능 |

**하나라도 해당되면 단정적 부정 표현 금지!**
→ "~도 있지만, ~도 있어서 상황에 따라 달라요"로 수정

#### 📋 직업운 특별 규칙
| 격국이 | 근데 신살에 | 그러면 |
|:-------|:-----------|:-------|
| 상관격 (자유로움) | 양인살/장성살 있음 | "조직 안 맞음" ❌ 금지 |
| 비겁 과다 (독립) | 협록 있음 | "혼자가 좋음" ❌ 금지 |
| 화개살 (고독) | 장성살 있음 | "혼자 일해야 함" ❌ 금지 |

---

| 확인 항목 | 보는 것 | 예시 |
|:----------|:--------|:-----|
| □ 재성/관성 | 이성운 기본 | 재성 약함 → 인연 늦게 |

| □ **상충점** | 서로 다른 방향 | 둘 다 언급! |

## 🔴 4. 상충 해석 공식

### 기본 공식

```
"[A 요소]라서 [A 특성]이 있지만,
[B 요소]도 있어서 [B 특성]도 있어요.
[상황/환경]에 따라 다르게 발현돼요."
```

### 실전 예시

| 상충 상황 | 해석 |
|:----------|:-----|
| 상관격 + 양인살 | "창의적 표현을 원하지만, 양인살이 있어서 군대/경찰 같은 명확한 조직에서도 빛나요" |
| 비겁 과다 + 협록 | "독립심이 강하지만, 협록이 있어서 협업할 때 시너지도 나요" |
| 화개살 + 장성살 | "예술적 감각과 고독함이 있으면서, 조직에서 성장하는 힘도 있어요" |
| 식상 강함 + 인성 약함 | "표현력은 넘치지만, 도움받기 어려워해요. 근데 협록이 있으면 협업은 잘해요" |

## 🔴 5. 월운 해석 규칙

### 절대 규칙
```
프롬프트의 월운 표 데이터 그대로 사용!
```

### ❌ 하면 안 되는 것
- "1월 = 寅월" 같은 공식 임의 적용
- 스킬 파일의 일반 이론으로 월운 계산
- 프롬프트 표와 다른 데이터 사용

### ✅ 해야 하는 것
- 프롬프트에 있는 월운 표의 간지/오행 그대로 사용
- 해당 월운과 일간의 관계로 십성 계산
- 월운 오행과 용신/기신 관계로 좋은 달/주의할 달 판단

---

## 🔴 6. 신살 해석 규칙

### 절대 규칙
```
프롬프트의 계산기 결과 신살만 해석!
```

### ❌ 하면 안 되는 것
- 스킬 파일 보고 신살 직접 계산해서 추가
- 계산기 결과에 없는 신살 언급
- 신살 위치(년지/월지/일지) 임의 판단

### ✅ 해야 하는 것
- 프롬프트의 "길신/양면신/흉신" 목록만 해석
- 스킬 파일은 해석 톤/방법 참고용으로만 사용
- 위치 정보도 프롬프트에 있는 것만 사용

## 📝 7. 해석 템플릿

### 각 카테고리 기본 구조

```markdown
### [카테고리명]

**1. 기본 성향** (일간 + 오행)
→ "[일간] 일간에 [오행] 과다/부족이라서 ~한 특성이 있어요"

**2. 십성으로 보면**
→ "[십성]이 강해서 ~한 경향이 있어요"

**3. 격국으로 보면**
→ "[격국]이라 ~한 면이 있어요"

**4. 신살로 보면**
→ "[신살]이 있어서 ~한 특성/환경도 잘 맞아요"

**5. ⭐ 종합하면** (상충점 포함!)
→ "~한 면도 있고 ~한 면도 있어서,
   [상황A]에서는 이렇고 [상황B]에서는 저래요"
```

## 💡 8. 실전 적용 예시

### 예시 1: 직업운 (신광민님 케이스)

**사주 데이터:**
- 상관격 (창의적, 자유로움 추구)
- 비겁 2.1개 (독립심 강함)
- 협록 신살 (협업 시너지)
- 양인살 (규칙과 질서에서 빛남)

**❌ 단정적 해석:**
"상관격이라 프리랜서가 맞아요. 조직은 안 맞아요."

**✅ 종합적 해석:**
```
당신은 상관격에 비겁도 강해서 독립적으로 일하는 걸 선호해요.
하지만 협록이 있어서 팀 프로젝트에서 시너지가 나기도 하고,
양인살이 있어서 명확한 규칙이 있는 조직에선 오히려 빛날 수 있어요.

→ 프리랜서로 하되, 팀 프로젝트 참여
→ 또는 조직 내에서 독립적 역할
→ 둘 다 가능해요!
```

### 예시 2: 연애운

**사주 데이터:**
- 재성 약함 (연애운 약함)
- 화개살 (고독, 외로움)
- 금여록 (좋은 배우자 만남)

**❌ 단정적 해석:**
"재성이 약하니 연애 어려워요."

**✅ 종합적 해석:**
```
재성이 0.3개로 약해서 연애가 쉽게 안 풀릴 수 있어요.
화개살도 있어서 혼자 있는 시간이 필요하고요.

하지만 금여록이 있어서 좋은 배우자를 만날 복은 있어요.
→ 늦게 시작하지만, 만나면 좋은 사람 만나요.
→ 급하게 하지 말고, 천천히 알아가세요.
```

## 🎯 9. 자주 상충하는 조합

### 1. 독립 vs 협업

| 독립 지향 | 협업 유리 | 통합 해석 |
|:---------|:---------|:---------|
| 비겁 과다 | 협록 | "독립심 강하지만, 협업 시 시너지" |
| 상관격 | 장성살 | "자유로움 추구하지만, 조직에서 성장" |
| 화개살 | 천주귀인 | "혼자 시간 필요하지만, 귀인 복 있음" |

### 2. 조직 적응

| 조직 부적합 | 조직 적합 | 통합 해석 |
|:-----------|:---------|:---------|
| 상관격 | 양인살 | "창의적이지만, 명확한 규칙에서 빛남" |
| 비겁 과다 | 관성 있음 | "독립적이지만, 책임감도 있음" |
| 식상 과다 | 정관 | "표현 중시하지만, 안정도 원함" |

### 3. 재물운

| 재물 불리 | 재물 유리 | 통합 해석 |
|:---------|:---------|:---------|
| 재성 약함 | 식상생재 | "타고난 재물운 약하지만, 노력으로 벌 수 있음" |
| 비겁 강함 | 협록 | "돈 새지만, 협업하면 돈 벌림" |
| 재성 없음 | 용신 금(재성 오행) | "재성 약하지만, 용신 활용하면 OK" |

## ✅ 10. 작성 시 체크리스트

**MD 작성 전:**
- [ ] 상충하는 요소가 있는지 확인
- [ ] 각 카테고리(직업/연애/재물)마다 종합 체크

**작성 중:**
- [ ] "~예요" 단정 말고 "~할 수 있어요" 가능성 열어두기
- [ ] 반대 요소 있으면 반드시 둘 다 언급
- [ ] "하지만" "동시에" 같은 연결어 사용

**작성 후:**
- [ ] 단일 요소로만 결론 내린 부분 없는지 재확인
- [ ] 상충점을 "양면성"으로 잘 설명했는지 확인
- [ ] 부정적 요소도 긍정적 활용법 제시했는지 확인

→ 늦게 시작하지만, 만나면 좋은 사람!

## 🎓 12. 고급 팁

### 1. 양면성을 장점으로 승화

**상충 → 다재다능함**
```
❌ "이것도 저것도 애매해요"
✅ "여러 환경에서 적응할 수 있어요"
```

### 2. 약점을 보완 전략으로

**부족함 → 보완법**
```
❌ "금이 없어서 결단력 없어요"
✅ "금이 0.25개로 부족하니, 용신 루틴(정리, 마무리)으로 채우세요"
```

### 3. 시간 축으로 설명

**현재 vs 미래**
```
❌ "재성 약해서 돈 못 벌어요"
✅ "지금은 재물운이 약하지만, 대운에서 재성이 오면 급상승해요"
```

- 단정적 해석 → 양면적 해석

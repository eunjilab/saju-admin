## 📌 핵심 작성 규칙 (Memory 기반)

### 인연수/배우자상 관련
- **프리미엄 인연수 표**: 외모/첫인상(음양) + 성격/유형(육합) 표 분리
- 표 아래 "💡 외모는 음양 오행, 성격은 일지 육합으로 도출" 필수
- 인연수 설명 시 "일지가 ○○인 사람과 잘 맞아요" 구체적 명시

### 만남시기 계산법 (⚠️ 중요 - 둘 기준 다름!)
- **인연수 만남시기** = 일간이 생하는 오행의 대표지지
  - 木→火/巳午, 火→土/辰戌 등
- **배우자상 만남시기** = 일지와 육합하는 지지
- ⚠ 둘 기준 겹치지 않게 주의!

### 점수 공식 (100점 만점)
| 항목 | 배점 | 세부 |
|------|------|------|
| ①오행궁합 | 40점 | 상생(내가生=40, 나를生=35) |
| ②음양밸런스 | 30점 | 다름=30, 같음=20 |
| ③육합관계 | 30점 | 육합=30, 삼합=25, 암합=18, 충=12 |

### 포맷팅 규칙
- "외모보다 첫인상" 문구는 [인연수_이미지_삽입] 바로 아래에만! 본문 중간 중복 금지
- "~님에게 드리는 마무리 메시지", "마무리 한마디" 같은 문구 금지 (파싱 오류)
- **테마 영역 박스(─── 필수)는 반드시 코드블록(\`\`\`)으로 감싸기!** 안 감싸면 파싱 안 됨

### 맞춤 질문 섹션 필수 순서
1. 😭 가장 큰 고민
2. 🔁 반복되는 패턴
3. Q1~Q15 중 체크된 것만
4. ✨ 프리미엄 맞춤질문

⚠️ 순서대로, 하나도 빠짐없이!

### 프리미엄 사주 보고서 포맷
- good_example 포맷 그대로 복사
  - 페이지 버전 주석
  - 태그 열고 닫기
  - 전이/페이지/마지막 구조
- 창의성은 **내용만**
- ⚠ 탭/아스키아트 금지

### 인연수/배우자상 추가 규칙
- 음양오행이 기본이지만, **용신 오행도 함께 가진 사람이면 더 좋다**고 언급
- 운세표/맞춤질문과 일관성 유지!

### 추가질문지 작성법
- "앞으로 잘 되는지" 류 질문엔 **전체 흐름/인과 관점 필수!**
- ❌ 결론만 X
- ⭕ 왜 좋아지는지 명리 근거 포함:
  - 역성, 합충파, 용신 등
  - 나이별 흐름
  - 인과별 흐름 상세히

### 📋 내부 분석 단계 (고객에게 안 보임)

**보고서 작성 전에 이 분석을 먼저 수행하세요:**

```
[내부 분석 - 고객용 결과지에 포함하지 않음]

1. 참조한 스킬 파일:
   □ saju-basics.md - 오행 분포 확인
   □ sipsung-guide.md - 십성 분포 확인
   □ hapchung-guide.md - 합충 관계 확인
   □ sinsal-guide.md - 신살 목록 확인
   □ daeun-guide.md - 현재 대운/세운 확인
   □ gyeokguk-guide.md - 격국/용신 확인

2. 이 사주의 핵심 구조:
   - 일간: ○○ (신강/신약)
   - 용신: ○○ / 기신: ○○
   - 주요 신살: ○○, ○○, ○○
   - 합충 관계: ○○
   - 특수 구조: 병존, 자형 등

3. 해석 원칙 확인:
   - 직접적 표현 ✓
   - 사주 근거 포함 ✓
   - 구체적 해결법 ✓

[분석 완료 → 고객용 결과지 작성 시작]
```

### ❌ 나쁜 예시 vs ✅ 좋은 예시

**❌ 스킬 파일 안 읽고 쓴 해석:**
> "건강에 주의가 필요해요."
> "이 시기는 조심하세요."
> "관계에 신경 쓰면 좋겠어요."

→ 두리뭉실, 근거 없음, 해결법 없음

**✅ 스킬 파일 참조해서 쓴 해석:**
> "8월에 심장이랑 혈압 조심하세요. 화(火) 기운이 과해지는 달이에요. (daeun-guide 참조)
> - 야근 줄이세요
> - 매운 음식, 술 피하세요
> - 물 하루 2리터 드세요"

> "6월에 돈 나갈 일 생겨요. 비겁운이라 그래요. (sipsung-guide 참조)
> - 보증 절대 서지 마세요
> - 투자 권유 다 거절하세요"

→ 구체적, 근거 있음, 해결법 있음

# 병존/반복 구조 해석 가이드

같은 글자가 여러 개 있거나, 특정 오행이 너무 많을 때 해석하는 법이에요.

---

## 1. 천간 병존 (같은 천간 2개 이상)

### 양간 병존

| 병존 | 해석 |
|:---:|:---|
| 갑갑 (甲甲) | 리더 둘이서 충돌, 고집 세고 타협 안 됨 |
| 병병 (丙丙) | 열정 과다, 너무 급함, 화재/열병 주의 |
| 무무 (戊戊) | 산이 두 개, 듬직하지만 융통성 없음 |
| 경경 (庚庚) | 칼이 두 개, 결단력 강하지만 무뚝뚝 |
| 임임 (壬壬) | 물 넘침, 지혜롭지만 우유부단 |

### 음간 병존

| 병존 | 해석 |
|:---:|:---|
| 을을 (乙乙) | 덩굴 엉킴, 유연하지만 의지 약함 |
| 정정 (丁丁) | 촛불 두 개, 섬세하지만 예민함 |
| 기기 (己己) | 포용력 있지만 욕심 많을 수 있음 |
| 신신 (辛辛) | 보석 두 개, 예민하고 까칠함 |
| 계계 (癸癸) | 비 계속 옴, 직관 좋지만 우울할 수 있음 |

### 병존이 있으면?
- **해당 오행 기운 2배** - 좋으면 더 좋고, 나쁘면 더 나쁨
- **비겁 과다 주의** - 일간과 같은 글자가 많으면 고집 셈
- **쟁합 발생** - 합 대상이 있으면 서로 차지하려고 경쟁

## 2. 지지 병존 (같은 지지 2개 이상)

### 일반 병존

| 병존 | 오행 | 해석 |
|:---:|:---:|:---|
| 자자 (子子) | 수 과다 | 감정 기복 심함, 신장/방광 주의 |
| 축축 (丑丑) | 토 과다 | 고집 셈, 소화기 주의 |
| 인인 (寅寅) | 목 과다 | 추진력 강함, 간/담 주의 |
| 묘묘 (卯卯) | 목 과다 | 유연함, 간 주의 |
| 사사 (巳巳) | 화 과다 | 열정적, 심장 주의 |
| 미미 (未未) | 토 과다 | 감성적, 소화기 주의 |
| 신신 (申申) | 금 과다 | 결단력, 폐 주의 |
| 술술 (戌戌) | 토 과다 | 고집 셈, 창고 열림 |

### 자형 (스스로 해치는 병존) ⚠️

**이 4개는 특히 조심!**

| 자형 | 해석 |
|:---:|:---|
| 진진 (辰辰) | 창고가 열림, 비밀 노출, 자기 비판 심함 |
| 오오 (午午) | 태양 두 개, 번아웃, 화병, 심장/혈압 |
| 유유 (酉酉) | 날카로움 배가, 예민/신경질, 자해적 성향 |
| 해해 (亥亥) | 물 넘침, 생각 과다, 우울/불안 |

**자형 있으면:**
- 스스로를 힘들게 하는 경향
- 해당 장기 건강 챙기기
- 생각 너무 많이 하지 말기

## 3. 오행 편중 (특정 오행이 너무 많거나 없거나)

### 오행 과다 (3개 이상)

| 오행 | 많으면 | 건강 주의 | 부족한 영향 |
|:---:|:---|:---|:---|
| 목 과다 | 성급, 분노, 계획만 많음 | 간/담 | 금 부족 → 결단력 약함 |
| 화 과다 | 열정적, 급함, 감정적 | 심장 | 수 부족 → 지혜 부족 |
| 토 과다 | 고집, 느림, 의심 많음 | 위장 | 목 부족 → 추진력 약함 |
| 금 과다 | 냉정, 비판적, 융통성 없음 | 폐 | 화 부족 → 열정 부족 |
| 수 과다 | 우유부단, 두려움, 감정적 | 신장 | 토 부족 → 안정감 부족 |

### 오행 부재 (0개)

| 없는 오행 | 의미 | 보완법 |
|:---:|:---|:---|
| 목 없음 | 시작/추진 약함 | 초록색, 동쪽, 봄 |
| 화 없음 | 열정/표현 약함 | 빨간색, 남쪽, 여름 |
| 토 없음 | 안정/신뢰 약함 | 노란색, 중앙 |
| 금 없음 | 결단/실행 약함 | 흰색, 서쪽, 가을 |
| 수 없음 | 지혜/유연 약함 | 검은색, 북쪽, 겨울 |

| 병오 (丙午) | 화 강함 | 열정, 급함 |
| 정사 (丁巳) | 화 강함 | 지혜, 예민 |
| 무진/무술 | 토 강함 | 신뢰, 고집 |
| 기축/기미 | 토 강함 | 포용, 현실적 |
| 경신 (庚申) | 금 강함 | 결단, 냉정 |
| 신유 (辛酉) | 금 강함 | 예민, 날카로움 |
| 임자 (壬子) | 수 강함 | 지혜, 감정적 |

## 5. 양인 일주 (강력한 일주들)

일지에 양인(강한 칼날)이 있는 경우

| 일주 | 양인 | 특징 |
|:---:|:---:|:---|
| 갑묘 (甲卯) | 묘 | 목 극강, 고집 매우 셈 |
| 병오 (丙午) | 오 | 화 극강, 열정적, 급함 |
| 무오 (戊午) | 오 | 열정 + 안정 |
| 경유 (庚酉) | 유 | 금 극강, 결단력 최강 |
| 임자 (壬子) | 자 | 수 극강, 지혜 + 감정 |

**양인 일주면:**
- 에너지가 매우 강함
- 승부욕, 리더십
- 날카로움 조절 필요

## 6. 쌍충/쌍합 구조

### 쌍충 (같은 충이 2번)

```
예: 자(子) - 오(午) - 자(子) - 오(午)
    ↑───↓     ↑───↓
    자오충     자오충
```

**의미:**
- 충의 효과 매우 강함
- 변동/이동 잦음
- 감정 기복 심함
- 건강 주의

### 쌍합 (같은 합이 2번)

```
예: 자(子) - 축(丑) - 자(子) - 축(丑)
    ↑───↓     ↑───↓
    자축합     자축합
```

**의미:**
- 합의 효과 매우 강함
- 인연/결합 기운 강함
- 해당 오행이 매우 강해짐

## 7. 실전 해석 예시

### 예시 1: 수 부재 사주

```
갑술 / 정묘 / 기유 / 경오

오행 분포:
- 목: 갑, 묘 = 2개
- 화: 정, 오 = 2개
- 토: 기, 술 = 2개
- 금: 경, 유 = 2개
- 수: 0개 ⚠️

해석:
"4개 오행이 균등한데 수(水)가 없어요.
→ 지혜, 유연성이 부족할 수 있음
→ 신장/방광 건강 챙기기
→ 수 보완: 검은색, 북쪽, 겨울 활용"
```

### 예시 2: 수 과다 사주

```
임자 / 임자 / 임자 / 갑인

병존 구조:
- 임 3개 (천간 삼병존)
- 자 3개 (지지 삼병존)

오행 분포:
- 수: 6개 = 극과다
- 목: 2개

해석:
"수(水)가 6개! 엄청 많아요.
→ 매우 지혜롭지만 우유부단할 수 있음
→ 감정 기복 심함
→ 신장/방광 건강 주의
→ 화(火), 토(土) 없음 → 열정, 안정감 부족"
```

## 8. 해석 꿀팁

1. **단순 개수만 보지 말기** - 위치와 관계도 중요해요
2. **계절(월지) 고려** - 태어난 달에 따라 강약 달라짐
3. **용신과 연결** - 과다 오행이 용신인지 기신인지가 핵심
4. **건강 연결** - 과다/부족 오행의 장기 챙기기
5. **자형 특별 주의** - 진진, 오오, 유유, 해해는 꼭 짚어주기

## 9. 오행별 건강 정리

| 오행 | 관련 장기 | 과다하면 | 부족하면 |
|:---:|:---|:---|:---|
| 목 | 간, 담, 눈, 근육 | 화 잘 냄, 경직 | 시력 약함, 피로 |
| 화 | 심장, 소장, 혀 | 심장 두근, 열 많음 | 냉증, 무기력 |
| 토 | 비장, 위장, 입술 | 소화불량, 비만 | 영양 부족 |
| 금 | 폐, 대장, 피부 | 피부 예민 | 호흡기 약함 |
| 수 | 신장, 방광, 귀 | 부종, 하체 약함 | 신장 약함 |

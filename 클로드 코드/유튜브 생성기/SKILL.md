# YouTube 영상 정보 추출기

YouTube 영상의 모든 정보를 추출하여 Markdown 파일로 저장하는 스킬입니다.

## 실행 순서 (Claude가 따라야 할 단계)

### 1단계: 스크립트 실행
```bash
/usr/bin/python3 "/Users/eunji/사주 관리자페이지/클로드 코드/유튜브 생성기/extract_youtube.py" "YOUTUBE_URL"
```

### 2단계: 결과 파일 읽기
- 출력에서 저장된 파일 경로 확인
- Read 도구로 해당 파일 읽기

### 3단계: 자막 전체 번역 (외국어인 경우)
- 자막이 영어 등 외국어면 한국어로 **전체 번역** (요약 아님)
- "## 자막/스크립트" 섹션의 내용을 한국어 전체 번역으로 교체
- 원본의 모든 내용을 빠짐없이 번역

### 4단계: AI 요약 작성 (상세하게)
- 자막과 설명을 바탕으로 영상 내용 **상세하게** 요약
- "## AI 요약 요청" 섹션을 "## AI 요약"으로 변경하고 요약 내용 작성
- 요약 형식:
  - 영상 개요 (2-3문장)
  - 주요 섹션별로 ### 제목과 bullet points로 상세 정리
  - 영상에서 언급된 도구, 리소스, 링크 정리
  - 핵심 인사이트나 팁 강조

### 5단계: 파일 업데이트
- Edit 도구로 번역된 자막과 AI 요약을 파일에 저장

## 예제 요청

- "이 유튜브 영상 정보 저장해줘: https://www.youtube.com/watch?v=xxxxx"
- "유튜브 영상 분석해줘 [URL]"
- "이 영상 자막 추출해서 저장해줘"

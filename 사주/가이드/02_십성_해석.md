# 십성 해석 가이드

십성은 "나(일간)"와 다른 글자들의 관계예요.
쉽게 말해 **"이 사람이 나한테 뭐지?"**를 알려줘요.

---

## 1. 십성 한눈에 보기

| 관계 | 같은 음양 | 다른 음양 | 쉬운 설명 |
|:---:|:---:|:---:|:---|
| 나와 같은 오행 | 비견 | 겁재 | 친구, 경쟁자 |
| 내가 낳는 오행 | 식신 | 상관 | 내 재능, 표현 |
| 내가 이기는 오행 | 편재 | 정재 | 돈, 재물 |
| 나를 이기는 오행 | 편관 | 정관 | 직장, 스트레스 |
| 나를 낳는 오행 | 편인 | 정인 | 공부, 도움 |

---

## 2. 십성별 쉬운 해석

### 비견 (나랑 똑같은 사람)

**한마디:** "내 친구이자 라이벌"

**좋을 때:**
- 혼자가 아니야, 함께할 동료가 있어
- 자립심 강함, 혼자서도 잘함
- 승부욕, 독립심

**나쁠 때:**
- 내 것 나눠가짐 (재물 분산)
- 고집 세서 타협 안 됨
- 경쟁자 많아짐

**이런 사람:** 형제, 친구, 동료, 경쟁자

---

### 겁재 (내 재물 빼앗는 사람)

**한마디:** "같은 편인 줄 알았는데 경쟁자"

**좋을 때:**
- 활동적, 사교성 좋음
- 배포 크고 스케일 있음
- 승부욕 강함

**나쁠 때:**
- 돈 쓸 일 많아짐
- 투기, 도박 성향
- 믿었던 사람에게 배신

**이런 사람:** 이성 형제, 경쟁자, 사업 파트너

**주의:** 겁재 많으면 "합작 사업 조심하세요"

---

### 식신 (내가 낳은 순한 자식)

**한마디:** "먹을 복, 표현력"

**좋을 때:**
- 먹을 복 있어요
- 낙천적, 여유로움
- 재능 있고 표현 잘함
- 건강하고 복 있음

**나쁠 때:**
- 게으를 수 있음
- 향락에 빠지기 쉬움

**이런 사람:** (여자) 딸, (남자) 장모

**"식신생재":** 식신 → 재성 연결되면 재능으로 돈 벎

---

### 상관 (내가 낳은 반항적 자식)

**한마디:** "날카로운 재능, 권위 싫어함"

**좋을 때:**
- 창의력 뛰어남
- 언변 좋고 표현력 강함
- 기술력, 전문성 있음

**나쁠 때:**
- 반항심, 윗사람과 마찰
- 직장에서 갈등 (상관이 정관을 해침)
- (여자) 남편복 약할 수 있음

**이런 사람:** (여자) 아들, (남자) 외할아버지

**주의:** "상관견관" - 상관이 정관 만나면 직장 갈등

---

### 편재 (내가 이기는 큰 돈)

**한마디:** "사업 돈, 아버지"

**좋을 때:**
- 사업 수완 있음
- 융통성 좋고 사교적
- 투자 감각, 재테크 잘함

**나쁠 때:**
- 투기, 도박 성향
- 낭비벽
- 바람기 (편재=여자친구 의미도)

**이런 사람:** 아버지, (남자) 애인/여자친구

---

### 정재 (내가 이기는 안정적 돈)

**한마디:** "월급, 아내"

**좋을 때:**
- 성실하게 돈 모음
- 검소하고 저축 잘함
- 안정적 수입

**나쁠 때:**
- 너무 아까워함, 인색
- 융통성 없음

**이런 사람:** (남자) 아내, (여자) 시아버지

**차이:** 편재는 "큰돈 왔다갔다", 정재는 "꼬박꼬박 월급"

---

### 편관 (나를 이기는 강한 힘) - 칠살이라고도 해요

**한마디:** "스트레스, 하지만 잘 쓰면 권력"

**좋을 때:**
- 리더십, 결단력
- 추진력 강함
- 카리스마, 권위

**나쁠 때:**
- 스트레스 심함
- 건강 문제 (과로)
- 관재, 소송

**이런 사람:** (남자) 아들, (여자) 애인

**핵심:** 편관은 "압박감"이에요. 잘 제어하면 권력, 못하면 스트레스

---

### 정관 (나를 이기는 정당한 힘)

**한마디:** "직장, 명예, (여자는) 남편"

**좋을 때:**
- 안정적 직장
- 명예, 사회적 인정
- 책임감, 품위

**나쁠 때:**
- 규제에 얽매임
- 융통성 없음, 답답

**이런 사람:** (남자) 딸, (여자) 남편

**차이:** 편관은 "무서운 상사", 정관은 "좋은 회사"

---

### 편인 (나를 낳는 특이한 도움)

**한마디:** "특수한 재능, 영적인 것"

**좋을 때:**
- 직관력 뛰어남
- 예술적, 창의적
- 철학, 종교에 관심
- 특수 기술

**나쁠 때:**
- 의존적, 게으름
- 외로움, 불안정
- 생각만 많고 행동 안 함

**이런 사람:** 계모, 스승, 멘토

**주의:** "효신탈식" - 편인이 식신 만나면 식신의 복을 빼앗음

---

### 정인 (나를 낳는 정통 도움)

**한마디:** "엄마, 공부, 자격증"

**좋을 때:**
- 학업 잘됨
- 어머니 덕, 귀인 도움
- 문서운 좋음 (자격증, 계약서)
- 인덕 있음

**나쁠 때:**
- 의존적, 나태
- 결정 못 함, 우유부단
- 엄마한테 너무 의지

**이런 사람:** 어머니, 스승, 선배

**차이:** 편인은 "특이한 스승", 정인은 "학교 선생님"

---

## 3. 십성 많고 적음

### 특정 십성이 3개 이상일 때

| 십성 | 많으면 |
|:---:|:---|
| 비겁 과다 | 고집 세고 협동 어려움, 재물 나눠가짐 |
| 식상 과다 | 말 많고 생각 많음, 직업 자주 바뀜 |
| 재성 과다 | 돈에 민감, 인색할 수 있음 |
| 관성 과다 | 스트레스 심함, 직장/규제에 치임 |
| 인성 과다 | 생각만 많고 행동 안 함, 게으름 |

### 특정 십성이 없을 때

| 십성 | 없으면 |
|:---:|:---|
| 비겁 없음 | 외롭고 형제 인연 약함, 혼자 해야 함 |
| 식상 없음 | 표현력 부족, (여자) 자녀 인연 약함 |
| 재성 없음 | 재물운 약함, 아버지 인연 약함 |
| 관성 없음 | 직장운 약함, (여자) 남편 인연 약함 |
| 인성 없음 | 학업 인연 약함, 어머니 인연 약함 |

---

## 4. 십성 좋은 조합

| 조합 | 의미 |
|:---|:---|
| 식신 + 정재 | 식신생재 - 재능으로 돈 벎 (최고!) |
| 정관 + 정인 | 관인상생 - 명예 + 학업 다 잡음 |
| 상관 + 편재 | 상관생재 - 기술로 큰돈 |
| 편관 + 정인 | 살인상생 - 스트레스를 지혜로 제어 |

---

## 5. 십성 나쁜 조합

| 조합 | 의미 |
|:---|:---|
| 상관 + 정관 | 상관견관 - 직장 갈등, 관재 |
| 비견 + 정재 | 비겁쟁재 - 재물 뺏김, 경쟁 |
| 편인 + 식신 | 효신탈식 - 식복 빼앗김 |
| 겁재 + 정재 | 겁재패재 - 큰 재물 손실 |

## 6. 실전 해석 예시

**예: 己土 일간**

```
시주: 庚(상관) / 午(편인)
일주: 己(나) / 酉(식신)
월주: 丁(편인) / 卯(편관)
연주: 甲(정관) / 戌(겁재)

십성 분포:
- 비겁: 1개 → 혼자 힘으로
- 식상: 2개 → 표현력, 재능 좋음
- 재성: 0개 → 재물은 노력으로
- 관성: 2개 → 직장운/스트레스
- 인성: 2개 → 학업운/생각 많음

해석:
"표현력 좋고(식상) 직장운 있지만(관성),
재물은 스스로 만들어야 해요(재성 없음).
생각이 많아서 행동력 필요해요(인성 과다)."
```

---

## 7. 해석 꿀팁

1. **일간 강약 먼저** - 신강이면 식상/재성이 좋고, 신약이면 인성/비겁이 좋아요
2. **없는 것에 집착 말기** - 없으면 대운/세운에서 채워져요
3. **조합 보기** - 단독보다 조합이 중요해요
4. **지지는 지장간으로** - 지지 속 숨은 천간까지 봐야 해요
